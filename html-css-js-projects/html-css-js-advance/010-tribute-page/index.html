Project 010: Tribute Page

Brendan Eich

Creator of JavaScript

Timeline of JavaScript Creation:

  • 1995 - Joined Netscape with the goal to add Scheme to the browser
  • May 1995 - Created JavaScript in 10 days under pressure from Netscape + Sun
  • 1995 - JavaScript first shipped in Netscape Navigator 2.0
  • 1996 - Submitted to ECMA for standardization, became ECMAScript
  • 2005 - AJAX + JS made web apps possible, sparking Web 2.0
  • 2009 - Node.js released, JavaScript goes to the server
  • 2015 - ES6 released, modern JS era begins
  • Present - Powers 98% of websites, React, Vue, Angular, and the modern web

"Always bet on JavaScript."

-- Brendan Eich
