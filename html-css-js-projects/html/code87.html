Code 87: <nav>

Note

nav purpose: Use <nav> for major navigation blocks only. Not every group of links needs nav — just primary navigation.

Multiple navs: Page can have several <nav> elements. Use aria-label to distinguish them: "Main", "Breadcrumb", "Footer".

What goes inside: Usually <ul>, <ol>, or plain <a> links. Can include headings for screen readers.

Not for all links: Don't wrap every link in nav. Footer links to terms/privacy are fine, but random inline links are not nav.

Accessibility: <nav> is a landmark. Screen reader users can jump to navigation regions with shortcuts.
