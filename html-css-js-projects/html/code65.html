Code 65: <label>

The <label> element represents a caption for a form control. Clicking the label focuses or activates the control.

Note

label rules: Always associate labels with controls. Use for="id" matching the input's id, or wrap the input inside the label.

Accessibility: Screen readers read the label when the input gets focus. Without labels, users won't know what to enter. Never use just placeholder.

Click behavior: Clicking a label triggers the control — focuses text inputs, toggles checkboxes/radios, opens selects. Increases hit area for touch.

Don't do this: <label>Email</label><input type="email"> with no for. The label isn't connected. Always link them.
