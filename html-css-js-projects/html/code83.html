Code 83: <summary>

The <summary> element defines a visible heading for a <details> element. Clicking it toggles the details content.

Note

summary rules: Must be first child of <details>. Only one <summary> per <details>. If missing, browser shows "Details".

Clickable area: The entire <summary> content is clickable. Browsers add a triangle/arrow marker by default.

Content allowed: Can contain phrasing content, headings, images, <strong>, <em>. Avoid interactive elements like buttons or links — clicking them toggles instead.

Accessibility: <summary> is the accessible name for the disclosure widget. Screen readers announce it as a button.

details required: <summary> only works inside <details>. Using it alone does nothing.
