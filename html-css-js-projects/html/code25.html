Code 25: <mark> Highlighted Text

The <mark> element represents text that is marked or highlighted for reference purposes, due to its relevance in another context.

What it does

<mark> is semantic highlighting. Think search results, exam notes, or quoted relevance.

Browsers typically render it with a yellow background — like a highlighter pen.


The Code

HTML 
<p>Search results for "HTML": Learn <mark>HTML</mark> basics.</p>
<p>The professor said <mark>this will be on the exam</mark>.</p>

Live Example

Search results for "HTML": Learn HTML basics in this tutorial.

The professor said this will be on the exam, so pay attention.

In the code review, she said: The variable userName should be camelCase.

Highlighting in quotes

The quick brown fox jumps over the lazy dog. The quick fox is very agile.

— Typing Example

mark vs em vs strong

mark vs em vs strong: Use <mark> for highlighting relevant text, like search matches. Use <em> for stress emphasis. Use <strong> for importance.

<mark> = relevance. <em> = inflection. <strong> = importance.
