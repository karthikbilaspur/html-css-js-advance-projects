Code 26: <small>

The <small> element represents side comments and small print like copyright and legal text. Not for making text visually smaller.

What it does

<small> is for disclaimers, legal text, attributions, or side comments.

It's semantic — it means "this is fine print," not "make this text tiny." Browsers usually render it one size smaller.


The Code

HTML 
<p>Price: $99.99 <small>Taxes not included</small></p>
<p><small>Copyright © 2026 HTML 100 Codes Vault. All rights reserved.</small></p>

Common Mistake

small usage: Use <small> for legal disclaimers, copyright, attribution, or side comments.

Don't use it just to make text smaller — that's a CSS job. <small> has meaning. If you just want visual size, use font-size.
