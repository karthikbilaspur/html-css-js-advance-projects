Code 30: <ins> Inserted Text

The <ins> element represents text that has been added to a document. It's the counterpart to <del> for showing edits.

What it does

<ins> marks content that was inserted after the document was published.

Used for changelogs, legal docs, collaborative editing, and showing revisions. Browsers usually render it with an underline.


The Code

HTML 
<p>Meeting rescheduled: <del>Monday</del> <ins>Tuesday at 3 PM</ins></p>
<p>Price updated: <del>$199</del> <ins datetime="2026-06-10">$149</ins></p>

ins vs u

ins vs u: Use <ins> for content added in an edit, like document revisions. Use <u> for unarticulated annotations like misspelled words.

Attributes: datetime shows when it was added: 2026-06-10. cite can link to an explanation of the change.
