Code 12: <p> Paragraph

What it does

The <p> tag defines a paragraph of text. Browsers add space above and below each paragraph automatically.

This is your main tag for blocks of text.


The Code

HTML 
<p>This is the first paragraph.</p>
<p>This is the second paragraph. Browsers add space between them.</p>

Live Example

This is paragraph one. Notice the spacing.

This is paragraph two. See how the browser separates them without you adding <br> tags?


Don’t do this

Don’t use multiple <br> tags to make space. Use separate <p> tags instead.
