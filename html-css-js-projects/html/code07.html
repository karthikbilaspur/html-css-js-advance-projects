Code 07: <body> Tag

What it does

The <body> tag contains everything visible on your webpage: text, images, links, videos, everything.

If it’s not in <head>, it goes in <body>.


The Code

HTML 
<body>
  <h1>This heading is visible</h1>
  <p>This paragraph is visible too.</p>
</body>

Key Rules

  • Only one <body> per page
  • Comes after </head>
  • Everything users see must be inside it
