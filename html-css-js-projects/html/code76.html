Code 76: <thead>

The <thead> element groups header content in a table. It must appear before <tbody> and contains header rows.

Examples

Basic thead with th

ID Name Score
101 Amit 88
102 Neha 94

thead with multiple rows

Employee Q1 Q2
Sales Target Sales Target
Raj 50k 45k 60k 55k

thead for printing

Item Qty Price
Book1₹200
Pen3₹30
Book1₹200
Pen3₹30
Book1₹200
Pen3₹30
Browsers repeat thead on each printed page

thead with scope

Month Revenue Growth
Jan ₹1.2L +5%

Note

thead rules: Must be direct child of <table>. Only one <thead> per table. Comes before <tbody> and <tfoot>.

Key purpose: Groups header rows. Browsers repeat <thead> at top of each page when printing long tables.

Content: Contains <tr> elements. Those <tr> should contain <th>, not <td>, for headers.

Accessibility: Using <thead> with <th scope="col"> helps screen readers identify column headers vs data.

Optional but recommended: Table works without <thead>, but adding it improves semantics, printing, and accessibility.
