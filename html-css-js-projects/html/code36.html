Code 36: <dd> Description Details

The <dd> element provides the description, definition, or value for a term <dt> in a description list <dl>.

What it does

<dd> is the "value" in a name-value pair. It’s the definition, description, or details.

Unlike <dt>, it can hold full block content — paragraphs, lists, images, even nested <dl>.


The Code

HTML 
<dl>
  <dt>CSS</dt>
  <dd>Cascading Style Sheets, used to style HTML documents.</dd>
</dl>

dd Rules

dd rules: <dd> must be inside <dl> and come after a <dt>.

Unlike <dt>, it can contain flow content — paragraphs, lists, images, even other <dl> elements.

Multiple <dd> can follow one <dt> for multiple definitions.

Key difference: <dt> = inline only. <dd> = blocks allowed.
