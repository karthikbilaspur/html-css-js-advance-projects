Code 3: The <head> Tag

What it does

The <head> section holds metadata about the page. Nothing here shows up on the page itself.

Think of it as the brain: it tells the browser the title, character set, and other behind-the-scenes info.


The Code

HTML 
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Your Page Title Here</title>
</head>

Common tags that go inside <head>

  1. <meta charset="UTF-8"> - Fixes text encoding
  2. <meta name="viewport"> - Makes site mobile-friendly
  3. <title> - Text shown in browser tab
  4. <meta name="description"> - Summary for Google
  5. <link> - Used for CSS, but we’re skipping that

Live Example
