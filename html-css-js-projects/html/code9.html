Code 9: <main> Element

What it does

<main> wraps the primary content of your page. Screen readers use it to jump straight to the important stuff.

Only use one <main> per page. Don’t put site-wide things like headers, footers, or nav inside it.


The Code

HTML 
<main>
  <h1>Blog Post Title</h1>
  <p>This is the actual content of the page.</p>
</main>

Live Example

All the explanations, code samples, and lists on this page are inside a <main> tag. The header and footer are outside it.
