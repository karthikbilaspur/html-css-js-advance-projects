Code 78: <tfoot>

The <tfoot> element groups footer content in a table. Usually contains summaries, totals, or results. Renders after <tbody> regardless of source order.

Examples

Basic tfoot with totals

Product Price
Book ₹250
Pen ₹30
Total ₹280

tfoot before tbody in source

Month Sales
Grand Total ₹95,000
January ₹45,000
February ₹50,000
tfoot renders at bottom even when written before tbody

tfoot with colspan

Item Qty Rate Amount
Keyboard 2 ₹600 ₹1200
Mouse 3 ₹300 ₹900
Total Amount ₹2100

tfoot for pagination

User Status
ArunActive
DivyaActive
Showing 1-2 of 50 users

Note

tfoot rules: Must be direct child of <table>. Only one <tfoot> per table. Contains <tr> elements with summary data.

Render order: Browser always displays <tfoot> after <tbody>, even if you write it before <tbody> in HTML.

Purpose: Use for totals, averages, summaries, or pagination info. When printing, browsers repeat <tfoot> at bottom of each page.

Optional: Table works without <tfoot>. Add it when you need footer rows semantically separate from body data.
