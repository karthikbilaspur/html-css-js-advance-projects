Code 6: <title> Tag

What it does

The <title> tag sets the text that shows up in the browser tab, bookmarks, and Google search results.

It goes inside <head> and every page should have exactly one.


The Code

HTML 
<title>Code 06: title tag - HTML 100 Codes</title>

Live Example

Look up at your browser tab right now. The text "Code 6: title tag - HTML 100 Codes" is coming from this page's <title> tag.

Good titles are short, descriptive, and unique per page.
