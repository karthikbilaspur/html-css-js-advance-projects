Code 80: <colgroup>

The <colgroup> element groups one or more columns in a table for formatting or semantics. Used with <col> elements.

Note

colgroup purpose: Groups columns for semantics or styling. Must be direct child of <table>, before <thead>, <tbody>, or <tr>.

Key elements: <colgroup> wraps <col> elements. Each <col> represents one column. Use span to make one <col> cover multiple columns.

Multiple groups: You can have multiple <colgroup> elements to create logical column sets. Or use span on <colgroup> itself.

No content: <col> is empty/self-closing. It doesn't contain cells. It just defines column structure for CSS or semantics.

Styling use: Even without inline CSS here, <col> lets you target entire columns with CSS selectors like col:nth-child(2).
