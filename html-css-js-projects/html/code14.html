Code 14: <hr> Horizontal Rule

What it does

<hr> creates a horizontal line across the page. It’s used to separate content sections.

It’s an empty tag — no closing tag needed. It represents a thematic break.


The Code

HTML 
<p>Section one content.</p>
<hr>
<p>Section two content.</p>

Live Example

Content above the line.

Content below the line. That line above? That’s an <hr>.


Pro Tip

In HTML5, <hr> means a "thematic break" between paragraphs. Don’t use it just for decoration — that’s what CSS border is for.
