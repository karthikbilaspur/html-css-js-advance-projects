Code 2: The <html> Tag

What it does

The <html> tag is the root element of every HTML page. Everything else goes inside it.

Adding lang="en" tells browsers and screen readers the page is in English. This helps with accessibility and SEO.


The Code

HTML 
<html lang="en">
  <head>
    ...metadata goes here...
  </head>
  <body>
    ...visible content goes here...
  </body>
</html>

Key Rules

  • Only one <html> per page
  • Always comes right after <!DOCTYPE html>
  • Must contain <head> and <body>
  • Set lang to your language: lang="hi" for Hindi, lang="es" for Spanish
