Code 29: <del> Deleted Text

The <del> element represents text that has been deleted from a document. Often used with <ins> to show edits.

What it does

<del> marks text as removed. It's semantic — screen readers can announce it as deleted.

Browsers typically render it with strikethrough. Perfect for changelogs, price drops, and edit tracking.


The Code

HTML 
<p>Current price: <del>$199</del> $149</p>
<p>The meeting is <del datetime="2026-06-09">Tuesday</del> <ins>Wednesday</ins>.</p>

Live Example

With cite and datetime attributes

This feature is experimental This feature is stable


del vs s

del vs s: Use <del> for text actually removed from a document, like edits or price changes.

Use <s> for text that's no longer accurate or relevant but not deleted. <del> = edit history. <s> = outdated.

Attributes: cite links to a document explaining the change. datetime gives when it was deleted, in ISO format.
