Code 64: <button>

The <button> element represents a clickable button. Can contain text, images, or other elements unlike <input type="button">.

Note

button type rules: Default type is submit inside forms. Always set type="button" if you don't want it to submit.

Key attributes: type="submit|reset|button" controls behavior. disabled prevents clicking. name and value submit with form. form links button to form by id.

button vs input: <button> can contain HTML like <strong> or <img>. <input type="button"> only takes plain text in value.

Accessibility: Buttons are keyboard focusable by default. Use for actions, not navigation. For links to pages, use <a> instead.
