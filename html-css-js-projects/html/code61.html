Code 61: <select>

The <select> element represents a dropdown control for selecting options. Contains <option> elements.

Note

select rules: <select> wraps <option> elements. The value of the selected option is submitted. If no value, the text content submits.

Key attributes: multiple allows selecting many options. size="4" shows it as listbox instead of dropdown. required forces selection if first option has empty value="".

Option control: selected makes an option default. disabled on option prevents selection. Use <optgroup label="..."> to group related options.

Submit behavior: Single select sends name=value. Multiple select sends name=val1&name=val2. Empty selection submits nothing unless first option has value="".
