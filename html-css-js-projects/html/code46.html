Code 46: <map>

The <map> element defines an image map. Used with <area> elements to create clickable areas on an image.

Examples

Image map with rectangular areas

Polygon area


Note

map + area pair: <map> requires a name attribute. The <img> links to it with usemap="#name". Inside <map> you define clickable <area> elements.

area shapes: rect uses x1,y1,x2,y2. circle uses x,y,radius. poly uses x1,y1,x2,y2,x3,y3... default covers everything else.

Accessibility: Always add alt on each <area>. Image maps are hard for keyboard users — provide text links as fallback.
