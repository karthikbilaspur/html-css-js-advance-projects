Code 55: <input type="number">

The <input type="number"> element creates a field for entering numbers. Browsers show spinner controls and numeric keyboards on mobile.

Note

number type rules: Only accepts floating-point numbers. Letters blocked by browser. Spinner arrows increment by step value.

Key attributes: min and max set range. step sets increment — use step="0.01" for currency, step="any" for any decimal. value sets default.

Validation: Blocks non-numbers and out-of-range values on submit. Empty is valid unless required. step mismatch shows error.

UX caveat: Spinner controls are tiny on mobile. For ZIP codes or IDs, prefer type="text" inputmode="numeric" pattern="[0-9]*" to get keypad without spinners.
