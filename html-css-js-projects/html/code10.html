Code 10: <footer> Element

What it does

<footer> is for content at the bottom of a page or section: copyright, contact info, related links.

Like <header>, you can have multiple — one for the whole page, and one per <article>.


The Code

HTML 
<footer>
  <p>&copy; 2026 My Site</p>
  <p><a href="contact.html">Contact Us</a></p>
</footer>

Note: &copy; renders as ©
