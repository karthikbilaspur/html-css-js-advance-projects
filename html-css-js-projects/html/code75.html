Code 75: <th>

The <th> element defines a header cell in a table. Use for row or column labels. Must be inside <tr>.

Note

th purpose: Use <th> for headers, not data. Browsers render it bold and centered by default. Screen readers announce "header" before content.

scope attribute: scope="col" for column headers. scope="row" for row headers. Critical for accessibility — tells users what each cell relates to.

abbr attribute: Provides abbreviated header name. Screen readers may use it instead of full text: <th abbr="Temp">Temperature</th>.

id + headers: For complex tables, give <th id="name"> then reference in cells: <td headers="name">. Explicitly links data to headers.

th vs td: Never use <td> for headers. Use <th> even if you don't want bold — style with CSS. Semantics matter.
