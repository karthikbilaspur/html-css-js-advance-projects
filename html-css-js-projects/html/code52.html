Code 52: <input>

The <input> element is used to create interactive controls for web-based forms. Its behavior changes based on the type attribute.

Note

input rules: <input> is a void element — no closing tag. The type attribute determines behavior. Default is type="text" if omitted.

Key types: text, email, password, url, tel, number, range, date, time, color, file, checkbox, radio, hidden, submit, reset, button.

Validation: Use required, min, max, minlength, maxlength, pattern. Browser blocks submit if invalid unless novalidate on form.

Accessibility: Always pair with <label for="id">. Use placeholder only as hint, not replacement for labels. radio inputs need same name to group.
