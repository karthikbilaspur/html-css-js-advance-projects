Code 5: <meta name="viewport">

What it does

The viewport meta tag controls how your page looks on mobile phones and tablets.

Without it, mobile browsers zoom out and make your site look tiny to fit the whole page on screen.


The Code

HTML 
<meta name="viewport" content="width=device-width, initial-scale=1.0">

Breaking it down:

  • width=device-width = Use the actual screen width of the device
  • initial-scale=1.0 = Don't zoom in or out by default

This tag also goes inside <head>.


Why it matters

Even with no CSS, this tag makes your HTML-only site readable on a phone. Try opening this page on mobile — text will be normal size instead of ant-sized.
