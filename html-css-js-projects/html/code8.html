Code 8: <header> Element

What it does

<header> is a semantic tag for introductory content. Usually holds a logo, site title, or main navigation.

You can have multiple <header> tags — one for the page, and one inside each <article> or <section>.


The Code

HTML 
<header>
  <h1>My Website Name</h1>
  <p>Welcome to my site</p>
</header>
