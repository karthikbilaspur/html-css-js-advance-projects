Code 19: <cite> Reference

The <cite> element represents the title of a creative work — book, paper, song, movie, etc. Not for people's names.

What it does

<cite> marks up titles of works you’re referencing. Browsers usually render it in italics.

It tells screen readers and search engines: “This is a title, not regular text.”


The Code

HTML 
<p>My favorite book is <cite>The Hobbit</cite> by J.R.R. Tolkien.</p>
<p>I learned HTML from <cite>MDN Web Docs</cite>.</p>

Live Example

My favorite book is The Hobbit by J.R.R. Tolkien.

I learned HTML from MDN Web Docs and it changed everything.

That quote comes from the movie The Matrix.

To be, or not to be, that is the question.

Common Mistake

cite usage: Use <cite> only for titles of works. Don't wrap author names in <cite> — that's a common mistake.

Wrong: <cite>J.R.R. Tolkien</cite>
Right: <cite>The Hobbit</cite> by J.R.R. Tolkien
