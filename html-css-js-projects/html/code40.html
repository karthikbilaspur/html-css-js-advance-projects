Code 40: <figcaption> Figure Caption

The <figcaption> element represents a caption or legend for its parent <figure> element.

What it does

<figcaption> gives context to a <figure>. It’s visible to everyone, not just screen readers.

Unlike alt which describes an image, <figcaption> describes the relationship or meaning.


The Code

HTML 
<figure>
  <img src="chart.png" alt="Bar chart showing 2026 traffic growth">
  <figcaption>Fig. 1 — Website traffic increased 40% in Q2 2026.</figcaption>
</figure>

figcaption Rules

figcaption rules: Must be inside <figure>. Can be the first or last child — position determines if caption shows above or below.

It describes the figure content for all users, not just screen readers. Use alt on <img> for accessibility; use <figcaption> for visible context.

Alt vs figcaption:

Use alt when Use figcaption when
Image fails to load You want visible caption
Screen reader needs description You reference it as "Fig. 1"
SEO needs image meaning You need attribution or context
