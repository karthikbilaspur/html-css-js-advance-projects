Code 50: <svg>

The <svg> element defines scalable vector graphics. SVG images stay crisp at any size and can be styled or scripted inline.

Note

svg rules: <svg> is an XML-based vector format. Unlike <canvas>, SVG shapes are DOM elements and remain accessible to CSS/JS.

Key attributes: width and height set display size. viewBox="0 0 100 100" sets the internal coordinate system. This lets SVG scale without distortion.

Common elements: <rect>, <circle>, <path>, <line>, <polyline>, <text>. Use fill, stroke, stroke-width for styling.

Accessibility: Add <title> and aria-labelledby for screen readers. Use role="img" if SVG is standalone. Inline SVG beats <img src="icon.svg"> for styling control.
