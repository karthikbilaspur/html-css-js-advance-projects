Code 62: <option>

The <option> element defines an item inside a <select>, <optgroup>, or <datalist>. It represents one choice.

Note

option rules: Must be inside <select>, <optgroup>, or <datalist>. Text between tags is what user sees. value is what submits.

Key attributes: value sets submitted data. If omitted, text content submits. selected makes it default. disabled prevents selection. label sets shorter dropdown text.

With select: Only one <option> can have selected unless parent <select multiple>. First option is default if none marked selected.

With datalist: <option> provides autocomplete suggestions for <input list="">. Use value only — text content ignored in most browsers.
