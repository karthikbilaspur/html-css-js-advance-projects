Code 13: <br> Line Break

What it does

<br> inserts a single line break. It’s an empty tag — no closing tag needed.

Use it when you need a line break but don’t want to start a whole new paragraph.


The Code

HTML 
<p>First line<br>
Second line<br>
Third line</p>

Live Example

Roses are red
Violets are blue
HTML is cool
And so are you


When NOT to use it

Don’t use <br><br><br> to add space between paragraphs. Use separate <p> tags instead.

Good for addresses and poems. Bad for layout spacing.
