Code 66: <legend>

The <legend> element represents a caption for the content of its parent <fieldset>. Must be the first child.

Legend must be first child of fieldset to be valid.


Note

legend rules: Must be direct child of <fieldset> and should be the first element. Browsers render it on the fieldset border.

Accessibility: Screen readers announce the legend text when a user enters any control in the fieldset. This is critical for radio/checkbox groups.

Content: Can contain inline elements like <strong>, <em>, <span>. Avoid block elements like <div> or <p> inside.

Without legend: A <fieldset> without <legend> is valid but less accessible. Always add one for grouped controls.
