Code 72: <table>

The <table> element represents tabular data — data arranged in rows and columns. Use only for actual data, not layout.

Examples

Basic table

Name Age City
Ravi 28 Bangalore
Priya 24 Delhi

Table with thead, tbody, tfoot

Product Qty Price
Mouse 2 ₹500
Keyboard 1 ₹1200
Total ₹2200

Table with caption

Monthly Sales 2025
Month Revenue
January ₹45,000
February ₹52,000

Table with colspan and rowspan

Team Q1 Results
Dev Jan 85%
Feb 90%

Table with scope attribute

HTML CSS
Arjun 95 88
Meera 90 94

Note

table structure: Use <tr> for rows. <th> for headers, <td> for data cells. Wrap headers in <thead>, data in <tbody>, totals in <tfoot>.

Key elements: <caption> gives table a title. colspan makes cell span columns. rowspan makes cell span rows.

Accessibility: Use <th scope="col"> for column headers, scope="row" for row headers. Screen readers use this to associate cells with headers.

Don't use for layout: Tables are for data only. For page layout use CSS Grid or Flexbox. Layout tables hurt accessibility and mobile.
