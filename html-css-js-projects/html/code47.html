Code 47: <area>

The <area> element defines a clickable region inside an image map. Must be a child of <map>.

Examples

Image map with area elements

Non-clickable area


Note

area rules: <area> is a void element — no closing tag. Only valid inside <map>. Requires alt for accessibility if href is present.

shape values: rect = x1,y1,x2,y2. circle = x,y,radius. poly = x1,y1,x2,y2,... default = rest of image.

Attributes: href sets link. coords sets clickable region. target, download, rel work like on <a>. Omit href to make a dead zone.
