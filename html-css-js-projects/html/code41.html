Code 41: <audio>

The <audio> element embeds sound content in documents. Use <source> for multiple formats.

Examples

Note

Key attributes: controls shows play/pause UI. autoplay starts immediately — most browsers block it unless muted is also set. loop repeats. preload="none|metadata|auto" controls loading. src or nested <source> elements set the file.

Fallback text: Content inside <audio> shows if the browser doesn't support it. Always include multiple <source> formats for browser compatibility: MP3 + OGG or AAC.
