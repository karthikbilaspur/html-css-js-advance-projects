Code 34: <dl> Description List

The <dl> element represents a description list. It encloses groups of terms with <dt> and descriptions with <dd>.

What it does

<dl> is for key-value pairs: glossaries, metadata, FAQs, specs.

It’s the most semantic way to show a term and its definition. Not just for dictionaries — think “Name: Value”.


The Code

HTML 
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language, the standard for web documents.</dd>
  
  <dt>CSS</dt>
  <dd>Cascading Style Sheets, used for styling HTML elements.</dd>
</dl>

dl Structure

dl structure: <dl> wraps the whole list. <dt> is the term/name. <dd> is the description/value.

You can group multiple <dt> for one <dd>, or multiple <dd> for one <dt>. Use it for glossaries, metadata, FAQs, or key-value pairs.
