Code 81: <col>

The <col> element defines column properties in a table. Must be inside <colgroup>. Used to apply attributes to entire columns.

Examples

Note

col rules: <col> is empty — no closing tag. Must be inside <colgroup>. Defines properties for columns left to right.

span attribute: <col span="3"> makes that col element apply to 3 columns. Default span is 1.

Limited styling: Only few CSS properties work on <col>: background, border, width, visibility. Text styles don't apply.

Modern approach: HTML5 made width, align, bgcolor obsolete. Use CSS with <col class="name"> instead.

colgroup required: <col> cannot be direct child of <table>. Always wrap in <colgroup> even if you have one column.
