Theme Switcher

Welcome to Code 3!

This is a beginner-friendly project that demonstrates how to toggle between light and dark themes using HTML, CSS, and JavaScript.

Click the button in the top right to switch between themes. Notice how the colors, shadows, and overall feel of the page changes smoothly.

What you'll learn:

  • CSS variables with :root and custom properties
  • Using classList.toggle() in JavaScript
  • Updating button text with textContent
  • Creating smooth transitions between themes
  • Building responsive layouts for mobile
