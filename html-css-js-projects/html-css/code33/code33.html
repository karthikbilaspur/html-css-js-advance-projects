Code 33: Image Carousel

Radio button slider with :checked - zero JavaScript

What it does

This carousel uses hidden <input type="radio"> elements and CSS sibling selectors (~). When a radio button is checked, CSS shows the corresponding slide and hides the others.

Labels act as dots/controls. Only one radio can be checked at a time because they share the same name.

The Code

<input type="radio" name="slider" id="s1" checked>
<input type="radio" name="slider" id="s2">

<div class="slides">
  <div class="slide">Slide 1</div>
  <div class="slide">Slide 2</div>
</div>

<label for="s1"></label>
<label for="s2"></label>
/* Hide all slides */
.slide { display: none; }

/* Show slide when radio is checked */
#s1:checked ~ .slides #slide1,
#s2:checked ~ .slides #slide2 {
  display: block;
}

Key Rules

  • All radios must share same name to form a group — only one checked at a time
  • Remove checked from all except the first input — you had all 3 checked
  • Hide radios with display: none or position: absolute; opacity: 0
  • Use general sibling selector ~ — radios must come before .slides in HTML
  • Style active dot with #s1:checked ~ .controls label[for="s1"]
  • Order matters: radios → slides → controls for ~ to work
  • No autoplay without JS — add it later if needed
