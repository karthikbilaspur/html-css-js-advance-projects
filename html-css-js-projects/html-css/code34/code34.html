Code 34: Before/After Slider

Image comparison with clip-path: inset() - zero JavaScript

What it does

This interactive component uses clip-path: inset(). By changing the inset values on :hover, we can dynamically "crop" the top image to reveal the one underneath.

Perfect for showing design changes, photo edits, or product transformations. No JS needed — pure CSS masking.

The Code

<div class="slider-container">
  <div class="image-before">Before</div>
  <div class="image-after">After</div>
</div>
.image-before {
  position: absolute;
  inset: 0;
  clip-path: inset(0 50% 0 0);
  transition: clip-path 0.3s;
}

.slider-container:hover .image-before {
  clip-path: inset(0 0 0 0);
}

Key Rules

  • Stack images with position: absolute inside relative container
  • clip-path: inset(top right bottom left) crops the element
  • inset(0 50% 0 0) shows left half, hides right half
  • Animate with transition: clip-path 0.3s for smooth reveal
  • On :hover change to inset(0 0 0 0) to show full image
  • Add a draggable range input for manual control — requires JS
  • For vertical slider, use inset(50% 0 0 0) to show top half
