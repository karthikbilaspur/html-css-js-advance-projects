Code 30: Mega Menu

Full-width dropdown with CSS Grid columns - zero JavaScript

What it does

A mega menu is a large dropdown that displays multiple columns of links, images, or featured content. Uses :hover + :focus-within to show, and CSS Grid for the multi-column layout.

Unlike simple dropdowns, mega menus can show entire site sections at once. Common on e-commerce and news sites.

Page Content Below

Hover over "Products" or "Solutions" to see the mega menu. The menu spans full width and uses CSS Grid for columns.

The Code

<li class="mega-dropdown">
  <a href="#">Products</a>
  <div class="mega-menu">
    <div class="mega-grid">
      <div class="mega-column">
        <h4>Category</h4>
        <ul><li><a href="#">Link</a></li></ul>
      </div>
    </div>
  </div>
</li>
.mega-dropdown:hover .mega-menu,
.mega-dropdown:focus-within .mega-menu {
  opacity: 1;
  visibility: visible;
}

.mega-grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 2rem;
}

Key Rules

  • Parent li needs position: relative or static with menu as position: absolute
  • Use width: 100% + left: 0 to make mega menu full-width
  • CSS Grid with grid-template-columns: repeat(4, 1fr) creates equal columns
  • Hide with opacity: 0 + visibility: hidden to allow transitions
  • :focus-within keeps menu open while tabbing through links
  • Add delay on mouse-leave with transition-delay to prevent accidental close
  • Mobile: convert to accordion or full-screen overlay — mega menus don't work on small screens
