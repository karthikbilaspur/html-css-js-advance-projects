Code 15: Newsletter Signup

What it does

This project builds a newsletter signup with email field + button combo using only HTML + CSS.

It demonstrates inline form layout with display: flex, :focus-within for group styling, and :valid/:invalid states for email validation.

The Code

<form class="newsletter-form">
  <div class="input-combo">
    <input type="email" placeholder="Enter email" required>
    <button type="submit">Subscribe</button>
  </div>
</form>

.input-combo {
  display: flex;
  border: 2px solid #e0e0e0;
}

.input-combo:focus-within {
  border-color: #667eea;
}

Key Rules

  • display: flex on wrapper - Makes input + button sit inline
  • :focus-within - Styles entire combo when input is focused
  • border-radius on input only left side, button only right side - Seamless combo look
  • input:invalid:not(:placeholder-shown) - Shows error only after user types
  • autocomplete="email" - Helps password managers and browsers autofill
  • Use flex: 1 on input so it takes remaining space, button stays fixed width
