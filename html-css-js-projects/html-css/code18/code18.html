Code 18: Custom Checkboxes

What it does

This project styles native checkbox inputs using appearance: none to remove browser defaults.

It demonstrates custom styling with ::before and ::after pseudo-elements, :checked state, and smooth transitions.

The Code

<label class="custom-checkbox">
  <input type="checkbox">
  <span class="checkmark"></span>
  Enable notifications
</label>

input[type="checkbox"] {
  appearance: none;
  width: 20px;
  height: 20px;
  border: 2px solid #ccc;
}

input[type="checkbox"]:checked {
  background: #667eea;
  border-color: #667eea;
}

Key Rules

  • appearance: none - Removes default browser checkbox styling
  • ::after or ::before - Creates custom checkmark with CSS
  • input:checked - Styles checkbox when selected
  • input:disabled - Styles disabled state with reduced opacity
  • Wrap in <label> - Makes entire area clickable, not just the box
  • transition on background/border - Smooth color change animation
