Code 37: Client Logos

Grayscale to color hover with CSS filter - zero JavaScript

What it does

This component uses the filter: grayscale(100%) property. On :hover, we set it back to grayscale(0%) to create a smooth, professional transition for brand logos.

Common on agency sites and SaaS landing pages to showcase clients without visual clutter.

Live Demo

↑ Hover over a logo to see color transition

The Code

<div class="logo-grid">
  <img src="logo1.svg" alt="Client 1" class="logo">
  <img src="logo2.svg" alt="Client 2" class="logo">
</div>
.logo {
  filter: grayscale(100%);
  opacity: 0.6;
  transition: filter 0.3s, opacity 0.3s;
}

.logo:hover {
  filter: grayscale(0%);
  opacity: 1;
}

Key Rules

  • filter: grayscale(100%) converts image to full grayscale
  • grayscale(0%) restores full color — use on :hover
  • Add transition on filter and opacity for smooth effect
  • Combine with opacity: 0.6 to make inactive logos more subtle
  • Use CSS Grid or Flexbox for .logo-grid layout
  • Works with SVG, PNG, JPG — but SVG scales best
  • Alternative: filter: brightness(0) invert(1) for monochrome logo effect
