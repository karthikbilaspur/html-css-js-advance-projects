Code 60: Timeline

Vertical line with events using ::before - zero JavaScript

What it does

Creates a vertical timeline for changelogs, project history, or event sequences. Uses ::before on the container to draw the central line and positioned markers for each event.

Perfect for company histories, product roadmaps, and resume sections. Fully semantic with <article> and <time>.

The Code

<div class="timeline">
  <article class="timeline-event">
    <div class="timeline-marker"></div>
    <div class="timeline-content">
      <time datetime="2026-01-15">January 15, 2026</time>
      <h3>Project Kickoff</h3>
      <p>Initial planning session...</p>
      <span class="timeline-tag">Milestone</span>
    </div>
  </article>
</div>

.timeline::before {
  content: '';
  position: absolute;
  left: 1rem;
  top: 0;
  bottom: 0;
  width: 2px;
  background: #ddd;
}

.timeline-marker {
  position: absolute;
  left: 0.5rem;
  width: 1rem;
  height: 1rem;
  background: #333;
  border-radius: 50%;
}

Key Rules

  • .timeline::before draws the vertical line with position: absolute
  • .timeline-marker uses border-radius: 50% for circular dots
  • position: relative on .timeline-event anchors the markers
  • margin-left on .timeline-content offsets text from the line
  • Use <article> for each event + <time datetime> for dates
  • .timeline-tag modifiers like --release color-code event types
  • On mobile, move line to left edge and reduce content offset
  • :last-child can fade the line end with a gradient
