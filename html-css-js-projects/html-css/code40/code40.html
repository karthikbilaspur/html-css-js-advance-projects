Code 40: Instagram Feed

3-column square grid with aspect-ratio - zero JavaScript

What it does

This layout mimics the Instagram profile grid. It uses display: grid with a 3-column repeat layout and the modern aspect-ratio: 1 / 1 property to keep every cell square.

No more padding hacks. aspect-ratio forces consistent squares regardless of image dimensions.

Live Demo

↑ 3 columns on desktop, responsive on mobile. All squares stay perfect 1:1

The Code

<div class="insta-grid">
  <div class="post"><img src="1.jpg" alt=""></div>
  <div class="post"><img src="2.jpg" alt=""></div>
  <div class="post"><img src="3.jpg" alt=""></div>
</div>
.insta-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 4px;
}

.post {
  aspect-ratio: 1 / 1;
}

.post img {
  width: 100%;
  height: 100%;
  object-fit: cover;
}

Key Rules

  • grid-template-columns: repeat(3, 1fr) creates 3 equal columns like Instagram
  • aspect-ratio: 1 / 1 forces square cells — replaces old padding-bottom: 100% hack
  • object-fit: cover on images prevents stretching while filling square
  • gap: 4px matches Instagram's tight spacing. Use gap: 1rem for desktop
  • For responsive: grid-template-columns: repeat(auto-fit, minmax(100px, 1fr))
  • Works in all modern browsers. Fallback: padding-bottom: 100% + position: absolute
  • Add :hover { transform: scale(1.02) } for interactive feedback
