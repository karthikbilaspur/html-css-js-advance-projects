Code 3: Recipe Card

What it does

This project creates a styled recipe card with image, ingredients list, and step-by-step instructions.

It uses semantic HTML like <article>, <ul>, <ol>, and CSS features like counter-increment for auto-numbered steps.

Live Demo

Classic Masala Dosa

⏱️ 45 mins 🍽️ Serves 2 🌶️ Medium

Ingredients

  • 1 cup Dosa batter
  • 2 Potatoes, boiled
  • 1 Onion, sliced
  • 1 tsp Mustard seeds
  • 1/2 tsp Turmeric powder
  • 2 tbsp Oil
  • Salt to taste

Instructions

  1. Heat oil in a pan. Add mustard seeds and let them splutter.
  2. Add onions and sauté till golden. Add turmeric and salt.
  3. Mash boiled potatoes and mix with onions. Masala is ready.
  4. Heat dosa tawa. Pour batter and spread in a circle.
  5. Drizzle oil around edges. Cook till golden and crisp.
  6. Place masala in center, fold dosa, and serve hot with chutney.

The Code

<article class="recipe-card">
  <img src="dosa.jpg" alt="Masala Dosa">
  <h3>Classic Masala Dosa</h3>
  <h4>Ingredients</h4>
  <ul>
    <li>1 cup Dosa batter</li>
  </ul>
  <h4>Instructions</h4>
  <ol>
    <li>Heat oil in a pan...</li>
  </ol>
</article>

Key Rules

  • Use <article> for self-contained recipe content
  • <ul> for ingredients, <ol> for steps
  • object-fit: cover keeps recipe images from stretching
  • Use ::marker or counter-increment to style list numbers
  • Wrap measurements in <span> to style them differently
