HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages and web applications. HTML describes the structure of a web page semantically.

CSS stands for Cascading Style Sheets. It describes how HTML elements should be displayed on screen, paper, or in other media. CSS saves a lot of work and can control multiple pages at once.

JavaScript is a programming language that enables interactive web pages. It is an essential part of web applications and can update content, control multimedia, animate images, and much more.

Responsive web design makes web pages render well on a variety of devices and window or screen sizes. It uses flexible layouts, media queries, and responsive images to adapt to the viewing environment.
