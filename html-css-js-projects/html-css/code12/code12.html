Code 12: Contact Form

What it does

This project builds a contact form with styled :focus, :valid, and :invalid states using only HTML + CSS.

It demonstrates HTML5 validation, CSS pseudo-classes for form feedback, and floating label effect with :placeholder-shown.

The Code

<div class="input-group">
  <input type="email" id="email" placeholder=" " required>
  <label for="email">Email Address</label>
  <span class="error-msg">Please enter a valid email</span>
</div>

input:invalid:not(:placeholder-shown) {
  border-color: #e74c3c;
}

input:valid:not(:placeholder-shown) {
  border-color: #27ae60;
}

Key Rules

  • :focus - Styles input when user clicks into it
  • :valid - Applies when input passes HTML5 validation
  • :invalid - Applies when input fails validation
  • :placeholder-shown - Detects if placeholder is visible/empty
  • Combine :invalid:not(:placeholder-shown) to only show errors after user types
  • pattern="[0-9]{10}" enforces 10-digit phone with regex
  • Use novalidate on form to disable browser popup, style with CSS instead
