Code 42: Features Grid

3-column icon cards with Grid - zero JavaScript

What it does

A classic SaaS landing page section. Uses CSS Grid for a 3-column layout with icon, heading, and description cards.

Perfect for showcasing product benefits, services, or value props. Auto-responsive with auto-fit.

The Code

<div class="grid-container">
  <div class="feature-card">
    <div class="icon">🚀</div>
    <h3>Fast Performance</h3>
    <p>Description</p>
  </div>
</div>
.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
  gap: 2rem;
}

.feature-card {
  text-align: center;
  padding: 2rem;
  border-radius: 8px;
  transition: transform 0.3s;
}

.feature-card:hover {
  transform: translateY(-8px);
}

Key Rules

  • repeat(auto-fit, minmax(280px, 1fr)) creates responsive 3→2→1 columns
  • text-align: center on cards for classic centered icon layout
  • Use emoji or SVG for .icon — set font-size: 3rem for emoji
  • Add aria-hidden="true" to decorative icons for accessibility
  • transform: translateY(-8px) on hover adds subtle lift effect
  • Keep card content short — 1-2 lines for h3, 2-3 lines for p
  • Add box-shadow or border to define cards on light backgrounds
