Code 5: Business Card

What it does

This project creates a digital visiting card with name, title, contact info, and logo using only HTML + CSS.

It demonstrates transform: rotate() for card flip effect on hover, CSS variables for theming, and responsive layout with flexbox.

The Code

<div class="business-card">
  <div class="card-front">
    <div class="logo">PS</div>
    <div class="company-name">PixelCraft Studio</div>
  </div>
  <div class="card-back">
    <h3>Priya Sharma</h3>
    <p class="title">Frontend Developer</p>
    <p>📧 priya@pixelcraft.in</p>
  </div>
</div>

Key Rules

  • Use perspective on parent + transform-style: preserve-3d for 3D flip
  • backface-visibility: hidden hides the back when front is showing
  • transform: rotateY(180deg) flips the card on :hover
  • Set fixed width and height like real cards: 350px x 200px
  • Use CSS variables --primary-color to easily theme the card
