Code 24: Breadcrumb Navigation

What it does

Breadcrumbs show users where they are in your site hierarchy. They use <nav> with aria-label="breadcrumb" for accessibility, and CSS ::after pseudo-elements for separators.

The last item is usually not a link, indicating the current page.

The Code

<nav aria-label="breadcrumb">
  <ol>
    <li><a href="#home">Home</a></li>
    <li><a href="#products">Products</a></li>
    <li aria-current="page">Current Page</li>
  </ol>
</nav>
/* Add separator after each li except last */
.breadcrumb li:not(:last-child)::after {
  content: "/";
  margin: 0 0.5rem;
  color: #999;
}

Key Rules

  • Use <nav aria-label="breadcrumb"> for screen readers
  • Structure with <ol> because order matters semantically
  • Mark current page with aria-current="page" and don't make it a link
  • Use ::after on li:not(:last-child) to add separators
  • Common separators: /, ›, →, or use border for arrows
  • Keep breadcrumb text short - truncate if needed on mobile
