Code 28: Multi-Column Responsive Footer

What it does

A footer with multiple link columns that stacks on mobile. Uses CSS Grid for responsive columns and <footer> semantic tag.

Common pattern: brand section + 3-4 link groups + social icons + copyright bar. Grid auto-flows to 1 column under 768px.

The Code

<footer class="site-footer">
  <div class="footer-container">
    <div class="footer-brand">
      <h3>Logo</h3>
    </div>
    <div class="footer-links">
      <h4>Links</h4>
      <ul>
        <li><a href="#">Home</a></li>
      </ul>
    </div>
  </div>
</footer>
.footer-container {
  display: grid;
  grid-template-columns: 2fr 1fr 1fr 1fr;
  gap: 2rem;
}

@media (max-width: 768px) {
 .footer-container {
    grid-template-columns: 1fr;
  }
}

Key Rules

  • Use <footer> semantic tag, not <div class="footer">
  • CSS Grid with grid-template-columns is easiest for responsive columns
  • Give brand column more space: 2fr 1fr 1fr 1fr
  • Use <ul> for link lists — semantically correct
  • Add aria-label to social icons if they only show symbols
  • Bottom bar with copyright should be separate from link grid
  • Stack to 1 column on mobile with @media (max-width: 768px)
