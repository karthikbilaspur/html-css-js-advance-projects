Code 13: Login Form

What it does

This project builds a login form with username/password and a "Show Password" toggle using only HTML + CSS.

It demonstrates the checkbox hack with :checked pseudo-class to toggle password visibility. No JavaScript required.

The Code

<div class="password-wrapper">
  <input type="password" id="password">
  <input type="checkbox" id="show-password">
  <label for="show-password">👁️</label>
</div>

#show-password:checked ~ #password {
  -webkit-text-security: none;
}

#show-password:checked ~ label .eye-open {
  display: none;
}

Key Rules

  • input[type="checkbox"]:checked ~ input[type="password"] - Checkbox hack to toggle state
  • -webkit-text-security: none - Webkit method to show password text
  • Fallback: Use type="text" instead of password when checked - needs JS
  • CSS-only limitation: -webkit-text-security only works in Chrome/Safari
  • autocomplete="username" and autocomplete="current-password" - Helps password managers
  • appearance: none - Removes default checkbox styling for custom design
