Code 10: Coming Soon Page

What it does

This project builds a "Coming Soon" page with animated countdown using only HTML + CSS.

It uses CSS @keyframes to animate numbers, animation-delay to stagger timers, and counter() is not used - we fake it with stacked animated spans. No JavaScript needed.

The Code

<div class="countdown">
  <div class="countdown-item">
    <div class="countdown-number">
      <div class="seconds-animation">
        <span>59</span>
        <span>58</span>
        <span>57</span>
      </div>
    </div>
    <div class="countdown-label">Seconds</div>
  </div>
</div>

Key Rules

  • CSS countdown is fake - it loops infinitely, not real time
  • @keyframes with transform: translateY() moves numbers up
  • animation: slide 60s steps(60) infinite - 1 step per second
  • Use overflow: hidden on number container to show one digit at a time
  • Real countdowns need JavaScript - this is for visual effect only
  • steps() timing function makes discrete jumps instead of smooth scroll
