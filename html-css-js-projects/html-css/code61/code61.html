Code 61: Dashboards & Data

Stat cards and data tables with CSS grid - zero JavaScript

What it does

Builds a dashboard layout with stat cards, trend indicators, and a data table. Uses CSS Grid for responsive layouts and utility classes for badges and status colors.

Perfect for admin panels, analytics pages, and internal tools. No JS charts needed for basic KPIs.

The Code

<section class="stat-grid">
  <div class="stat-card">
    <h3>Active Users</h3>
    <p class="stat-number">2,847</p>
    <span class="stat-trend up">+12.5%</span>
  </div>
</section>

<table class="data-table">
  <thead>
    <tr><th>User</th><th>Status</th></tr>
  </thead>
  <tbody>
    <tr>
      <td>@user</td>
      <td><span class="badge success">Complete</span></td>
    </tr>
  </tbody>
</table>

Key Rules

  • display: grid + repeat(auto-fit, minmax(200px, 1fr)) for responsive cards
  • .stat-trend.up/.down uses color + icons to show positive/negative change
  • border-collapse: collapse on tables for clean grid lines
  • .badge classes for status pills: .success, .warning, .error
  • overflow-x: auto on table wrapper prevents mobile layout break
  • Use <thead> + <tbody> for semantic tables and sticky headers
  • text-align: right on numeric columns for easier scanning
  • Keep stat numbers large: font-size: 2rem+ for hierarchy
