Introduction

Welcome to the Lumina Ring API documentation. Use our REST API to access health data, manage users, and build integrations with the Lumina ecosystem.

Base URL: https://api.lumina.dev/v1

Quick Start

Get your API key from the developer dashboard and make your first request:

curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.lumina.dev/v1/user/profile

Installation

Install our official SDKs for faster development:

Node.js

npm install @lumina/sdk

Python

pip install lumina-ring

Authentication

All API requests must include your API key in the Authorization header.

Authorization: Bearer lm_sk_1234567890abcdef
Keep your API keys secure. Do not expose them in client-side code or public repositories.

Rate Limits

Plan Requests/min Requests/day
Free 60 1,000
Pro 600 100,000

Health Data API

Retrieve sleep, activity, and heart rate data for authenticated users.

GET /data/sleep

Returns sleep data for a date range.

Query Parameters

Parameter Type Description
start_date string ISO 8601 date: YYYY-MM-DD
end_date string ISO 8601 date: YYYY-MM-DD

Example Response

{
  "data": [
    {
      "date": "2026-06-17",
      "duration": 28800,
      "deep_sleep": 7200,
      "rem_sleep": 5400,
      "score": 85
    }
  ]
}

Error Handling

The API uses standard HTTP status codes to indicate success or failure.

Code Meaning
200 Success
401 Unauthorized - Invalid API key
429 Too Many Requests - Rate limit exceeded
500 Internal Server Error
