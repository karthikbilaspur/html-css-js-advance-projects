Code 6: Product Card

What it does

This project builds a product card with image, title, price, and "Add to Cart" button using only HTML + CSS.

It demonstrates object-fit: cover for images, position: absolute for badges, and transition + transform for hover effects.

The Code

<article class="product-card">
  <div class="product-image">
    <img src="watch.jpg" alt="Smart Watch">
    <span class="badge">Sale</span>
  </div>
  <div class="product-info">
    <h3>Smart Watch Pro</h3>
    <div class="price">
      <span class="price-new">₹3,999</span>
      <span class="price-old">₹5,999</span>
    </div>
    <button class="btn-cart">Add to Cart</button>
  </div>
</article>

Key Rules

  • Wrap image in a div with overflow: hidden to contain zoom effect
  • Use object-fit: cover so images fill the space without stretching
  • position: absolute on badges to place them over images
  • transform: scale() on :hover for image zoom
  • Use text-decoration: line-through for old price
