Code 46: CTA Banner

Gradient call-to-action section - zero JavaScript

What it does

A full-width call-to-action banner that sits between sections. Uses a gradient background, centered content, and dual buttons to drive conversions.

Place this after features/pricing to push users to sign up or start a trial.

The Code

<section class="cta-banner">
  <div class="cta-content">
    <h2>Ready to Ship Faster?</h2>
    <p>Join 10,000+ developers...</p>
    <div class="cta-buttons">
      <button class="btn-primary">Start Free Trial</button>
      <button class="btn-secondary">View Demo</button>
    </div>
  </div>
</section>
.cta-banner {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  color: white;
  padding: 4rem 2rem;
  text-align: center;
}

.cta-content {
  max-width: 700px;
  margin: 0 auto;
}

.cta-buttons {
  display: flex;
  gap: 1rem;
  justify-content: center;
  flex-wrap: wrap;
}

.btn-primary, .btn-secondary {
  padding: 0.75rem 2rem;
  border-radius: 6px;
  transition: transform 0.2s;
}

.btn-primary:hover {
  transform: translateY(-2px);
}

Key Rules

  • linear-gradient(135deg, ...) creates angled brand gradient backgrounds
  • text-align: center + margin: 0 auto centers content block
  • Use flex-wrap: wrap on button group so they stack on mobile
  • gap: 1rem spaces buttons without margin hacks
  • Primary button: solid bg. Secondary: border: 2px solid white + transparent bg
  • transform: translateY(-2px) on hover gives tactile lift
  • Keep copy short: 1 line h2, 1-2 lines p, 2-3 words per button
