Code 21: Responsive Navbar

Hamburger menu with <details>/<summary> - zero JavaScript

What it does

This navbar uses the <details> and <summary> tags to create a hamburger menu with zero JavaScript. The browser handles the open/close state natively.

On screens wider than 768px you get a normal horizontal menu. On mobile, it collapses into a hamburger icon.

The Code

<details class="nav-toggle">
  <summary class="hamburger">
    <span class="bar"></span>
    <span class="bar"></span>
    <span class="bar"></span>
  </summary>
  <nav class="nav-menu">
    <a href="#home">Home</a>
    <a href="#about">About</a>
    <a href="#work">Work</a>
  </nav>
</details>

Key Rules

  • Use list-style: none on <summary> to hide the default triangle
  • Target details[open] in CSS to style the open state
  • Hide .nav-toggle on desktop with display: none
  • Position .nav-menu absolutely so it drops down over content
  • Add aria-label to summary for accessibility
  • Use @media (max-width: 768px) to toggle mobile/desktop views
