Code 9: 404 Error Page

What it does

This project builds a creative "Page Not Found" design using only HTML + CSS.

It demonstrates CSS animations with @keyframes, large typography, and centered layouts with flexbox. No JavaScript needed.

The Code

<div class="error-container">
  <div class="error-code">
    <span>4</span>
    <span>0</span>
    <span>4</span>
  </div>
  <h3>Oops! Page Not Found</h3>
  <p>The page you're looking for...</p>
  <a href="/" class="btn">Go Home</a>
</div>

Key Rules

  • Use display: flex + align-items: center to center vertically
  • @keyframes with transform: translateY() creates floating animation
  • Large font-size with text-shadow makes the 404 stand out
  • clamp() for responsive font sizes: clamp(4rem, 15vw, 8rem)
  • Always provide a "Go Home" link - users need an escape route
