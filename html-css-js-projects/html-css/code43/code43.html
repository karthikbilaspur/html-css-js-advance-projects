Code 43: Pricing Table

What it does

This project builds a responsive 3-tier pricing table. It features a highlighted "Pro" plan in the center, achieved by scaling the container and adding a distinct border/shadow.

Key Rules

  • Use grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)) for responsiveness.
  • Apply transform: scale(1.05) to the middle tier to make it stand out.
  • Use flex-direction: column on the cards to align content vertically.
  • Use z-index to ensure the popular card sits above others if they overlap.
