Code 4: Quote Card

This project displays a big quote with author attribution on a gradient background.

What it does

It uses <blockquote> and <cite> for semantic quotes, CSS linear-gradient for backgrounds, and ::before for decorative quote marks.

Live Demo

The best way to predict the future is to create it.

— Peter Drucker

Dream, dream, dream. Dreams transform into thoughts and thoughts result in action.

— Dr. A.P.J. Abdul Kalam

Be the change that you wish to see in the world.

— Mahatma Gandhi

The Code

<blockquote class="quote-card">
  <p>The best way to predict the future is to create it.</p>
  <cite>— Peter Drucker</cite>
</blockquote>

Key Rules

  • <blockquote> is the semantic tag for long quotes
  • <cite> is used for the author/source of the quote
  • linear-gradient() creates smooth color transitions for backgrounds
  • ::before with content: " " adds decorative quote marks
  • Use font-size with rem or clamp() for responsive text
