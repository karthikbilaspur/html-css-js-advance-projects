Code 23: Dropdown Menu with :hover & :focus-within

What it does

This dropdown menu works on mouse hover and keyboard focus with zero JavaScript. The :focus-within pseudo-class keeps the menu open when you Tab through links, making it accessible.

The dropdown is hidden by default and only shows when the parent li is hovered or has focus inside it.

The Code

<li class="dropdown">
  <a href="#services">Services</a>
  <ul class="dropdown-menu">
    <li><a href="#web">Web Design</a></li>
    <li><a href="#app">App Development</a></li>
  </ul>
</li>
/* Show on hover or keyboard focus */
.dropdown:hover.dropdown-menu,
.dropdown:focus-within.dropdown-menu {
  opacity: 1;
  visibility: visible;
  transform: translateY(0);
}

Key Rules

  • Parent li.dropdown needs position: relative
  • Child .dropdown-menu uses position: absolute to overlay
  • Hide with opacity: 0 + visibility: hidden not display: none to allow transitions
  • :focus-within is critical for keyboard users - keeps menu open while tabbing
  • Add slight transition for smooth fade/slide effect
  • Use pointer-events: none when hidden to prevent accidental hover
