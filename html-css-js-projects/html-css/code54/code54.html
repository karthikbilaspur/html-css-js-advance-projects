Code 54: Article Card Grid

Image, category, title & excerpt layout

What it does

Responsive grid of article cards for blogs and magazines. Each card includes a featured image with overlay category, linked title, excerpt, and metadata with date + read time.

Uses CSS Grid with auto-fit and object-fit: cover for consistent layouts.

The Code

<article class="article-card">
  <div class="card-image">
    <img src="..." alt="...">
    <span class="card-category">Design</span>
  </div>
  <div class="card-content">
    <h2 class="card-title"><a href="#">Title</a></h2>
    <p class="card-excerpt">Excerpt...</p>
    <div class="card-meta">
      <time datetime="2026-06-15">June 15, 2026</time>
    </div>
  </div>
</article>

Key Rules

  • display: grid + repeat(auto-fit, minmax(320px, 1fr)) = responsive cards
  • aspect-ratio: 16/10 on .card-image prevents layout shift
  • object-fit: cover keeps images consistent without distortion
  • position: absolute on .card-category overlays badge on image
  • Wrap <time datetime=""> for SEO + machine-readable dates
  • Use gap: 2rem on grid instead of margins for clean spacing
  • -webkit-line-clamp: 3 truncates excerpts uniformly
