Code 7: Book Card

What it does

This project creates a book card with cover image, title, author, and CSS-only star rating.

It uses ::before with content: "★★★★★" for stars, background-clip: text for gradient stars, and hover effects with transform.

The Code

<article class="book-card">
  <img src="cover.jpg" alt="Book Cover" class="book-cover">
  <h3>Atomic Habits</h3>
  <p class="author">by James Clear</p>
  <div class="rating" data-rating="5">
    <span class="stars"></span>
  </div>
</article>

Key Rules

  • Use data-rating="5" attribute to set star rating from 1-5
  • ::before with content: "★★★★★" creates the stars
  • background: linear-gradient + background-clip: text colors filled stars
  • aspect-ratio: 2/3 keeps book covers proportional
  • transform: translateY(-5px) on :hover adds lift effect
