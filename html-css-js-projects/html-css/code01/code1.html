Code 1: Tribute Page

What it does

This project builds a single-page tribute using only HTML and CSS. No JavaScript needed.

It demonstrates semantic HTML5 tags like <main>, <figure>, <blockquote>, and responsive design with CSS media queries.


Live Demo

Dr. A.P.J. Abdul Kalam

Missile Man of India & The People's President

Timeline of Dr. Kalam's life:

  • 1931 - Born in Rameswaram, Tamil Nadu
  • 1960 - Joined DRDO as a scientist
  • 1980 - SLV-III successfully deployed Rohini satellite
  • 1997 - Awarded Bharat Ratna
  • 2002 - Became the 11th President of India
  • 2015 - Passed away while delivering a lecture at IIM Shillong

"Dream, dream, dream. Dreams transform into thoughts and thoughts result in action."

— Dr. A.P.J. Abdul Kalam

The Code

<main>
  <h1>Dr. A.P.J. Abdul Kalam</h1>
  <figure>
    <img src="kalam.jpg" alt="Dr. Kalam">
    <figcaption>The People's President</figcaption>
  </figure>
  <ul>
    <li><strong>1931</strong> - Born in Rameswaram</li>
  </ul>
  <blockquote>
    <p>"Dream, dream, dream..."</p>
  </blockquote>
</main>

Key Rules

  • Use <figure> + <figcaption> for images with captions
  • Set max-width: 100% on images for responsiveness
  • Use <strong> for dates in timeline lists
  • <blockquote> is semantic for quotes
  • Keep CSS in a separate file and link it with <link rel="stylesheet">
