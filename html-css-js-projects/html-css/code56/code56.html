Code 56: Sticky Table of Contents

Sidebar TOC with position sticky - zero JavaScript

What it does

Documentation-style layout with sticky sidebar TOC. The TOC uses position: sticky to stay visible while scrolling through long content. Click links jump to sections via anchor IDs.

Essential for docs, guides, and long-form articles. No JS scroll-spy needed.

Modern SCSS Architecture Guide

Build maintainable stylesheets that scale with your team and project complexity.

Introduction

SCSS has become the de facto standard for writing maintainable CSS at scale. Its features like nesting, variables, and mixins help teams avoid repetition and enforce consistency.

Project Setup

Start with a clear folder structure. Separate concerns into partials: _variables.scss, _mixins.scss, _base.scss, and component files.

Core Concepts

Understanding SCSS fundamentals is critical before adopting advanced patterns.

Variables

Store colors, spacing, and breakpoints in variables. This creates a single source of truth for your design system.

Mixins

Mixins let you define reusable chunks of styles. Use them for vendor prefixes, media queries, or complex animations.

Nesting

Nest selectors to mirror your HTML structure, but avoid going deeper than 3 levels.

Advanced Patterns

Once comfortable with basics, explore functions, loops, and conditionals. @each loops can generate utility classes from maps.

Performance Tips

Avoid @extend in large projects — it can generate unexpected selector combinations. Prefer mixins for reusable styles.

Conclusion

SCSS is powerful, but with power comes responsibility. Establish conventions early, document your architecture, and refactor mercilessly.

The Code

<aside class="toc-sidebar">
  <nav class="toc-nav" aria-label="Table of contents">
    <h3 class="toc-title">On This Page</h3>
    <ul class="toc-list">
      <li><a href="#intro">Introduction</a></li>
    </ul>
  </nav>
</aside>

Key Rules

  • position: sticky + top: 2rem keeps TOC visible during scroll
  • align-items: start on grid prevents sidebar stretching to full height
  • max-height: calc(100vh - 4rem) + overflow-y: auto scrolls long TOCs
  • Link href="#id" to matching id="intro" on sections
  • scroll-behavior: smooth on html enables smooth anchor jumps
  • scroll-padding-top prevents sticky header from covering jumped content
  • Nest <ul class="toc-sublist"> for H4 subsections under H3
  • Use aria-label="Table of contents" for screen readers
