Code 22: Slide-in Sidebar with Checkbox

What it does

This sidebar uses a hidden <input type="checkbox"> and the :checked CSS selector to slide in/out. No JavaScript needed.

Click the hamburger to open, click the overlay or close icon to shut it. The checkbox state controls everything.

Page Content

Click the hamburger icon top-left. The sidebar slides in from the left and pushes an overlay over this content.

This pattern is used by YouTube, Discord, and most mobile apps.

The Code

<input type="checkbox" id="sidebar-toggle">
<label for="sidebar-toggle">☰</label>

<aside class="sidebar">
  <nav>
    <a href="#home">Home</a>
    <a href="#about">About</a>
  </nav>
</aside>

<label for="sidebar-toggle" class="overlay"></label>

Key Rules

  • Checkbox must come before the elements it controls in HTML
  • Use label for="id" to toggle checkbox from anywhere
  • Target with #sidebar-toggle:checked ~ .sidebar using the general sibling selector
  • Hide checkbox with display: none or opacity: 0
  • Overlay should also be a label so clicking it closes the menu
  • Use transform: translateX() for smooth GPU-accelerated animation
