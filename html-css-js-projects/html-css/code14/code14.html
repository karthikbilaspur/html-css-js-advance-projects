Code 14: Sign Up Form

What it does

This project builds a multi-field sign up form with nice layout using CSS Grid.

It demonstrates display: grid for 2-column layout, grid-column: span 2 for full-width fields, and password strength validation with HTML5 attributes.

The Code

<form class="signup-form">
  <div class="form-row">
    <div class="form-group">
      <label for="firstname">First Name</label>
      <input type="text" id="firstname" required>
    </div>
    <div class="form-group">
      <label for="lastname">Last Name</label>
      <input type="text" id="lastname" required>
    </div>
  </div>
</form>

.form-row {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 15px;
}

Key Rules

  • display: grid + grid-template-columns: 1fr 1fr creates 2-column layout
  • grid-column: span 2 or .full-width makes field span both columns
  • minlength="8" + pattern attribute for password validation
  • gap: 15px adds consistent spacing between grid items
  • Use autocomplete attributes: given-name, family-name, email
  • Wrap checkboxes in labels for better click area and accessibility
