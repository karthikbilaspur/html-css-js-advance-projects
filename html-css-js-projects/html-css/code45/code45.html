Code 45: Stats Counter

What it does

This creates a CSS-only number counter. By using @keyframes to animate a CSS variable, we can force the browser to recalculate and display increasing numbers.

Live Demo

0+ Projects
0+ Clients

Key Rules

  • Use @property to define a custom CSS variable as an integer for animation.
  • counter-reset is used alongside content: counter(variable) to render the dynamic number.
  • animation-fill-mode: forwards ensures the counter stays at the final number.
  • This technique is highly performant as it relies on CSS hardware acceleration.
