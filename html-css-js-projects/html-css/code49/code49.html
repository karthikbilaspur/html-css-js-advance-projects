Code 49: Brand Partners Strip

What it does

This creates a seamless, infinite scrolling strip of brand logos. It uses CSS @keyframes to animate the horizontal position of a container containing duplicated sets of logos.

Live Demo

Key Rules

  • Use white-space: nowrap on the container to prevent items from wrapping to a new line.
  • The @keyframes animation moves the margin-left or transform: translateX from 0 to -50%.
  • Ensure the total width of the logos is accounted for so the animation reset is invisible to the user.
