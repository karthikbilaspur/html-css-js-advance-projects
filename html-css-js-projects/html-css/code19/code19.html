Code 19: Custom Radio Buttons

What it does

This project builds custom radio button toggles for gender selection and theme select using only HTML + CSS.

It demonstrates appearance: none to remove default radio styling, :checked pseudo-class for active states, and label wrapping for larger click areas.

Live Demo

Profile Settings

The Code

<label class="radio-card">
  <input type="radio" name="gender" value="male">
  <span class="radio-content">
    <span class="radio-icon">👨</span>
    <span>Male</span>
  </span>
</label>

.radio-card input {
  appearance: none;
  position: absolute;
}

.radio-card input:checked + .radio-content {
  border-color: #667eea;
  background: #f0f4ff;
}

Key Rules

  • appearance: none - Removes default browser radio styling
  • position: absolute on input - Hides radio but keeps it accessible
  • input:checked + span - Styles the visual content when selected
  • Wrap input in label - Makes entire card clickable, better UX
  • Use name="gender" on all radios - Groups them so only one selects
  • transition on border/background - Smooth state changes
