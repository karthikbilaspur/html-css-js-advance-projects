Code 59: Code Snippet Box

Styled code blocks with headers - zero JavaScript

What it does

Upgrades boring <pre><code> blocks into professional code snippets. Adds a header bar with language + filename, syntax highlighting via CSS classes, and line numbers with counters.

Essential for docs, tutorials, and technical blogs. No JS syntax highlighters required for basic theming.

Live Demo

Styling Code Blocks for Readability

How to make your <pre><code> blocks look professional without JavaScript.

When writing technical content, code examples need to be scannable and copy-friendly. Default browser styling for <pre> is functional but ugly.

With pure CSS we can add line numbers, syntax-aware colors, a header bar, and hover states. No JS libraries needed for basic styling.

SCSS _mixins.scss 
// Responsive breakpoint mixin
@mixin breakpoint($size) {
  @if $size == tablet {
    @media (min-width: 768px) { 
      @content; 
    }
  }
  @else if $size == desktop {
    @media (min-width: 1024px) { 
      @content; 
    }
  }
}

The example above uses CSS counters for line numbers and semantic <span> classes for token colors. You can adapt the color scheme to match your brand.

The Code

<div class="code-snippet">
  <div class="code-header">
    <span class="code-language">SCSS</span>
    <span class="code-filename">_mixins.scss</span>
  </div>
  <pre><code>
@mixin breakpoint(...) {
  @content;
}
  </code></pre>
</div>

.code-snippet {
  background: #1e1e1e;
  border-radius: 8px;
  overflow: hidden;
}

.code-header {
  background: #2d2d2d;
  padding: 0.75rem 1rem;
  display: flex;
  justify-content: space-between;
}

.token-keyword { color: #c678dd; }
.token-string { color: #98c379; }

Key Rules

  • Wrap <pre><code> in a .code-snippet container for styling
  • .code-header uses flex to space language + filename
  • overflow-x: auto on <pre> enables horizontal scroll on mobile
  • Use counter-reset + ::before on lines for CSS-only line numbers
  • .token-* classes let you color syntax: .token-keyword, .token-string
  • font-family: 'Fira Code', monospace for ligatures and readability
  • white-space: pre preserves formatting, tab-size: 2 sets tab width
  • Dark theme: #1e1e1e bg. Light theme: #f6f8fa bg + modifier class
