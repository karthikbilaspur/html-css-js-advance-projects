Code 44: Testimonials Slider

CSS-only carousel with radio inputs - zero JavaScript

What it does

This creates a functional slider without JavaScript. By linking labels to hidden radio inputs, we can toggle visibility of specific slide content.

The radio hack: :checked + sibling combinators control which testimonial shows. Fully keyboard accessible.

The Code

<input type="radio" name="testimonial" id="s1" checked>
<input type="radio" name="testimonial" id="s2">

<div class="slides">
  <div class="slide">Testimonial 1</div>
  <div class="slide">Testimonial 2</div>
</div>

<label for="s1"></label>
<label for="s2"></label>
.slides { overflow: hidden; }

.slide {
  opacity: 0;
  position: absolute;
  transition: opacity 0.5s;
}

#s1:checked ~ .slides .slide:nth-child(1),
#s2:checked ~ .slides .slide:nth-child(2) {
  opacity: 1;
  position: relative;
}

input[type="radio"] { display: none; }

Key Rules

  • Hidden input[type="radio"] acts as the state manager — group with same name
  • Use ~ sibling combinator: #s1:checked ~ .slides .slide:nth-child(1)
  • overflow: hidden on container ensures only one slide visible
  • Hide inactive slides with opacity: 0 + position: absolute
  • Show active slide with opacity: 1 + position: relative to set height
  • transition: opacity makes switching smooth — no layout shift
  • Labels are the controls — style as dots. Add aria-label for accessibility
