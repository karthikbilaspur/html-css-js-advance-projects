Code 38: Video Thumbnail Grid

Play button overlay with absolute positioning - zero JavaScript

What it does

This layout uses absolute positioning to center a "Play" button over thumbnail images. The play button is hidden or transparent by default and becomes prominent on hover, providing immediate visual feedback for video content.

Pattern used by YouTube, Vimeo, and course platforms. Signals clickable video content.

The Code

<a href="#" class="video-card">
  <img src="thumb.jpg" alt="Video">
  <div class="play-btn">▶</div>
</a>
.video-card {
  position: relative;
  display: block;
}

.play-btn {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
  opacity: 0;
  transition: opacity 0.3s;
}

.video-card:hover .play-btn {
  opacity: 1;
}

Key Rules

  • Parent .video-card needs position: relative to contain absolute children
  • Center play button with top: 50%; left: 50%; transform: translate(-50%, -50%)
  • Hide button with opacity: 0, show on :hover with opacity: 1
  • Add overlay: .video-card::before with background: rgba(0,0,0,0.4) on hover
  • Use aria-hidden="true" on decorative play icon if link has accessible text
  • Wrap in <a> for keyboard access — add :focus styles matching :hover
  • For scale effect: transform: translate(-50%, -50%) scale(1.1) on hover
