Code 16: Feedback Form

What it does

This project builds a feedback form with interactive star rating using radio buttons and pure CSS.

It demonstrates the "checkbox hack" with input[type="radio"], :checked + ~ sibling selector, and flex-direction: row-reverse for hover effects.

The Code

<div class="star-rating">
  <input type="radio" id="star5" name="rating" value="5">
  <label for="star5">★</label>
  <input type="radio" id="star4" name="rating" value="4">
  <label for="star4">★</label>
  <!-- Repeat for 3, 2, 1 -->
</div>

.star-rating {
  flex-direction: row-reverse;
}

.star-rating input:checked ~ label {
  color: #ffc107;
}

Key Rules

  • flex-direction: row-reverse - Reverses stars so 5 is on left visually
  • input:checked ~ label - Colors all stars before the checked one
  • input:hover ~ label - Shows hover effect on previous stars
  • Hide radio with display: none, style labels as clickable stars
  • Use name="rating" on all radios so only one can be selected
  • ~ general sibling selector targets all following siblings
