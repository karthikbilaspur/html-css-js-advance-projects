Project 010: Tribute Page

Brendan Eich

Creator of JavaScript

Timeline of JavaScript

  • April 1995
    Joined Netscape

    Hired to embed Scheme in the browser. Plans changed quickly.

  • May 1995
    Created JavaScript in 10 days

    Built under pressure from Netscape + Sun to ship with Navigator 2.0

  • December 1995
    JavaScript ships in Netscape 2.0

    Initially called Mocha, then LiveScript, finally JavaScript

  • November 1996
    Submitted to ECMA

    Standardized as ECMAScript to avoid trademark issues

  • 2005
    AJAX & Web 2.0

    JS + XMLHttpRequest enabled Gmail, Google Maps, and modern web apps

  • May 2009
    Node.js released

    Ryan Dahl brings JavaScript to the server using Chrome's V8

  • June 2015
    ES6 / ES2015

    Classes, modules, arrow functions. Modern JS era begins

  • Present
    Powers 98% of websites

    React, Vue, Angular, Svelte. Deno, Bun. Still evolving

"Always bet on JavaScript."

Brendan Eich
