# Code1 to Code10

Built 10 bite-sized, beginner-friendly projects to master DOM manipulation, events, and core JS concepts. Each project is standalone with clean HTML/CSS/JS and progressive navigation.

Perfect for anyone starting out in frontend dev. No frameworks, just vanilla JS fundamentals.

📚 What I Built:
Code 1: Hello JS Alert
First dive into JavaScript. Learned alert(), console.log(), and DOM ready events. The "Hello World" of interactive web pages.
Stack: alert(), console.log(), getElementById

Code 2: Change Text on Click
Making pages interactive. Used addEventListener() to update DOM content on button click.
Stack: getElementById(), addEventListener(), DOM manipulation

Code 3: Dark/Light Mode Toggle
Theme switcher using CSS variables and classList.toggle(). Smooth transitions + persistent UI states.
Stack: CSS :root variables, classList.toggle(), textContent

Code 4: Live Character Counter
Real-time input tracking with input events. Counter changes color as you approach max length. Great UX practice.
Stack: maxlength, input event, conditional styling

Code 5: Password Toggle
Show/hide password feature. Toggle input.type between password and text. Small detail, big UX win.
Stack: Input type switching, aria-label, button states

Code 6: Random Background Color
Click to generate random RGB values. Learned about Math.random() and dynamic style updates.
Stack: Math.random(), style.backgroundColor, template literals

Code 7: Digital Clock
Live ticking clock using setInterval(). Working with Date object and formatting time.
Stack: setInterval(), Date(), toLocaleTimeString()

Code 8: Simple Calculator
Basic calc with onclick handlers. Evaluates expressions and handles clear functionality.
Stack: onclick, eval(), disabled inputs, event handling

Code 9: Image Hover Zoom
CSS + JS hover effects. Scale images on mouseover for interactive galleries.
Stack: mouseover/mouseout, CSS transform: scale(), transitions

Code 10: Accordion FAQ
Collapsible FAQ section. Toggle answer visibility with smooth expand/collapse. Common UI pattern mastered.
Stack: classList.toggle(), event delegation, max-height transitions

🎯 Key Takeaways
Vanilla JS first: Master the fundamentals before jumping to frameworks
Small wins compound: 10 tiny projects > 1 complex unfinished project
DOM is king: If you can manipulate the DOM, you can build anything
Progressive navigation: Each project links to the next. Learning should feel like a journey
