Code 49: Brand Partners Strip

Infinite scrolling logos with keyframes - zero JavaScript

What it does

This creates a seamless, infinite scrolling strip of brand logos. It uses CSS @keyframes to animate the horizontal position of a container containing duplicated sets of logos.

Perfect for "Trusted by" sections. Loops forever with no JS, no libraries.

The Code

<div class="logo-strip">
  <div class="logo-slide">
    <span>Brand A</span>
    <span>Brand B</span>
    <!-- Duplicate all logos once -->
    <span>Brand A</span>
    <span>Brand B</span>
  </div>
</div>

Key Rules

  • white-space: nowrap on container prevents logos from wrapping
  • Duplicate the logo set once — this creates the illusion of infinite scroll
  • @keyframes moves transform: translateX(0) to translateX(-50%)
  • Animate to -50% because half the content is duplicated
  • Use animation: scroll 20s linear infinite for constant speed
  • overflow: hidden on parent hides logos outside viewport
  • Add animation-play-state: paused on :hover to pause on interaction
  • Use will-change: transform for GPU acceleration on long strips
