Code 43: Pricing Table

3-tier responsive pricing with highlighted plan - zero JavaScript

What it does

This project builds a responsive 3-tier pricing table. It features a highlighted "Pro" plan in the center, achieved by scaling the container and adding a distinct border/shadow.

The middle tier stands out to drive conversions — a standard SaaS pattern.

The Code

<div class="plan popular">
  <h3>Pro</h3>
  <p class="price">$30<span>/mo</span></p>
  <ul><li>Feature 1</li></ul>
  <button>Choose Plan</button>
</div>
.pricing-container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 2rem;
  align-items: center;
}

.plan.popular {
  transform: scale(1.05);
  border: 2px solid #3b82f6;
  box-shadow: 0 8px 24px rgba(0,0,0,0.15);
  z-index: 10;
}

.plan {
  display: flex;
  flex-direction: column;
}

Key Rules

  • grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)) for responsive tiers
  • transform: scale(1.05) on middle tier makes it stand out without layout shift
  • z-index: 10 ensures popular card sits above others when scaled
  • flex-direction: column on cards lets you push button to bottom with margin-top: auto
  • align-items: center on container vertically centers non-scaled cards
  • Use ::before or .badge span for "Most Popular" ribbon
  • Wrap /mo in <span> to style it smaller than price
