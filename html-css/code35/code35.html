Code 35: Masonry Gallery

Pinterest-style layout with CSS columns - zero JavaScript

What it does

This layout uses column-count to create a masonry-style effect. Unlike standard Grid, this approach allows items to fill columns top-to-bottom, like a newspaper layout.

Items with different heights pack together without gaps. Perfect for photo galleries and Pinterest-style feeds.

Live Demo

↑ Images flow top-to-bottom, then left-to-right. Resize to see columns adjust

The Code

<div class="masonry-gallery">
  <img src="img1.jpg" alt="" class="brick">
  <img src="img2.jpg" alt="" class="brick">
  <img src="img3.jpg" alt="" class="brick">
</div>
.masonry-gallery {
  column-count: 3;
  column-gap: 1rem;
}

.brick {
  break-inside: avoid;
  margin-bottom: 1rem;
  width: 100%;
}

Key Rules

  • column-count: 3 creates 3 vertical columns. Use columns: 3 200px for responsive
  • break-inside: avoid on items prevents them from splitting across columns
  • column-gap controls horizontal spacing between columns
  • Use margin-bottom on items for vertical spacing
  • Items fill columns top-to-bottom, not left-to-right like Grid
  • Add column-width: 250px instead of count for truly responsive columns
  • Downside: DOM order is vertical, so tab order is column 1, then 2, then 3
