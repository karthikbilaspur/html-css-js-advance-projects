Code 36: Team Members Grid

Card layout with hover overlay - pure CSS transitions

What it does

This layout uses CSS Grid to create a responsive card system. Each card features an image and a social overlay that appears on hover, utilizing absolute positioning and transitions.

Common pattern for "Meet the Team" sections. Grid handles responsiveness, overlay handles interactivity.

The Code

<div class="member-card">
  <img src="person.jpg" alt="Name">
  <div class="overlay">
    <h3>Name</h3>
    <p>Role</p>
  </div>
</div>
.team-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 2rem;
}

.member-card {
  position: relative;
  overflow: hidden;
}

.overlay {
  position: absolute;
  inset: 0;
  opacity: 0;
  transition: opacity 0.3s;
}

.member-card:hover .overlay {
  opacity: 1;
}

Key Rules

  • display: grid with auto-fit + minmax() for responsive cards
  • Card needs position: relative so overlay can be position: absolute
  • overflow: hidden clips overlay and prevents scrollbars during transition
  • inset: 0 is shorthand for top: 0; right: 0; bottom: 0; left: 0
  • Hide overlay with opacity: 0 + transition for fade effect
  • Use transform: translateY() on overlay for slide-up animation
  • Add alt text to images and aria-label to icon-only social links
