Code 47: App Download Section

What it does

This layout displays a marketing section for mobile applications. It uses a split-screen design (text/badges on one side, mockup container on the other) which is standard in modern SaaS landing pages.

Key Rules

  • Use display: flex to align text content and the phone mockup side-by-side.
  • media queries are essential here to stack the items on mobile (switching from row to column).
  • The phone-mockup uses specific border and border-radius to visually represent a device.
  • Adding gap ensures clean spacing between elements without needing heavy margins.
