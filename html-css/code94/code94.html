Code 94: Advanced Image Carousel

Advanced Image Carousel

CSS-only carousel with prev/next controls and ARIA carousel pattern

Overview

This carousel uses hidden <input type="radio"> elements, CSS sibling selectors, and :has() for prev/next navigation. When a radio is checked, CSS shows the active slide and updates ARIA states. No JavaScript required.
