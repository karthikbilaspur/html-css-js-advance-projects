Code 32: CSS Lightbox

Modal image viewer with :target - zero JavaScript

What it does

This Lightbox effect uses the CSS :target pseudo-class. When you click a thumbnail, the URL hash changes to match the ID of the full-size image container, making it visible.

The overlay and close button work with pure CSS. Clicking the background or "×" changes hash to # to close it.

The Code

<a href="#img1">Thumbnail</a>

<div id="img1" class="lightbox">
  <a href="#" class="close">×</a>
  <div class="content">Full Image</div>
</div>
.lightbox {
  display: none;
  position: fixed;
  inset: 0;
  background: rgba(0,0,0,0.9);
}

.lightbox:target {
  display: flex;
}

Key Rules

  • Link href="#id" targets element with matching id
  • :target pseudo-class applies only when element’s ID matches URL hash
  • Hide lightbox by default with display: none, show with :target { display: flex }
  • Use position: fixed; inset: 0 for full-screen overlay
  • Close link uses href="#" to clear hash and hide lightbox
  • Add aria-label="Close lightbox" on close button for accessibility
  • No JS needed, but Esc key won't close — that requires JavaScript
