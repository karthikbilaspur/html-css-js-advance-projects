Code 58: Drop Cap Paragraph

Editorial drop caps with ::first-letter - zero JavaScript

What it does

Creates classic editorial drop caps where the first letter of a paragraph is enlarged and styled. Uses ::first-letter pseudo-element to target just the initial character without extra HTML.

Perfect for long-form articles, blogs, and magazine-style layouts. Fully accessible and semantic.

The Evolution of Web Typography

June 15, 2026 · 4 min read

Typography on the web has undergone a renaissance in the last decade. What was once limited to a handful of web-safe fonts has exploded into a universe of variable fonts, fluid sizing, and expressive layouts that rival print design.

The journey began with @font-face support, which freed us from Arial and Times New Roman. Suddenly brands could use their actual typefaces online. But that freedom came with performance costs — large font files and FOUT made early web fonts a compromise.

Today we have font-display, variable fonts, and system font stacks that give us the best of both worlds: personality and performance. Drop caps like the one above are now trivial with CSS, no images required.

Modern CSS gives us ::first-letter and ::first-line pseudo-elements to target specific parts of text. Combined with modern layout techniques, we can recreate classic print styles while keeping everything accessible and semantic.

Alternative Style

Elegant drop caps don't have to be traditional. By combining background colors, borders, and modern CSS, you can create contemporary variants that fit any design system while maintaining that editorial feel.

↑ First letters are styled automatically with CSS only

Key Rules

  • ::first-letter targets only the first character of a block element
  • float: left makes text wrap around the enlarged letter
  • font-size: 3-4rem + line-height: 0.8-1 prevents layout shift
  • margin controls spacing between drop cap and body text
  • Only works on block elements — add display: block to inline parents
  • Won't work if first character is in a <span> or has punctuation before it
  • Add font-family to use a different typeface for the drop cap
  • Combine with ::first-line for small-caps or bold first lines
