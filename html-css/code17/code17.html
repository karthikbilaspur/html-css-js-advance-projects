Code 17: Search Bar

Expand on focus with :focus-within and CSS transitions - pure HTML + CSS

What it does

This project builds a search bar that expands when focused using only HTML + CSS.

It demonstrates :focus-within to style the parent wrapper, transition for smooth width animation, and ::placeholder styling.

The Code

<div class="search-wrapper">
  <span class="search-icon">🔍</span>
  <input type="search" placeholder="Search...">
  <button>Go</button>
</div>

.search-wrapper {
  width: 200px;
  transition: width 0.4s ease;
}

.search-wrapper:focus-within {
  width: 400px;
}

Key Rules

  • :focus-within - Styles parent when any child input is focused
  • transition: width 0.4s ease - Smooth expand animation
  • type="search" - Gives mobile keyboards search button + clear X
  • ::placeholder - Style placeholder text color and opacity
  • flex: 1 on input - Makes input take remaining space in flex container
  • Hide button initially with width: 0, show on :focus-within for icon-only start
