Code 31: Photo Gallery Grid

Responsive grid with auto-fit + minmax() - no media queries

What it does

This layout demonstrates a fluid gallery that adapts to any screen size without needing media queries. By leveraging CSS Grid, we create a responsive design that automatically reconfigures based on available width.

The Code

<div class="gallery">
  <div class="photo">1</div>
  <div class="photo">2</div>
  <div class="photo">3</div>
</div>
.gallery {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 1rem;
}

Key Rules

  • repeat(auto-fit, minmax(200px, 1fr)) creates responsive columns without media queries
  • auto-fit: Collapses empty tracks. Items stretch to fill the row
  • minmax(200px, 1fr): Minimum 200px width, grows to fill available space equally
  • Use gap for spacing instead of margins — cleaner and no collapsing
  • Works for any number of items — grid auto-wraps to new rows
  • Replace auto-fit with auto-fill to keep empty columns for fixed track count
