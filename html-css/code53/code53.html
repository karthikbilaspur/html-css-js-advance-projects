Code 53: Magazine Layout

Multi-column text flow with CSS columns - print aesthetic

What it does

Magazine layouts rely on negative space and visual flow. By utilizing the CSS columns property, we force text to flow naturally between columns, much like a printed publication.

Excellent for long-form content where readability is priority. No JS, no Grid, no Flexbox.

Live Demo

The Art of Typography

Magazine layouts rely heavily on negative space and visual flow. The CSS columns property lets text flow naturally between columns, much like a printed publication.

This approach is excellent for long-form content where readability is a priority. Unlike traditional sidebar layouts, this design keeps the reader's eye moving down the page in a continuous rhythm. The text automatically balances across columns based on available space.

You can adjust column-gap to control white space between columns and column-rule to add a subtle separator. For headlines, use column-span: all to break out of the column flow.

Modern browsers handle multi-column layouts well, automatically calculating optimal line lengths. Combine this with hyphens: auto and text-align: justify for a true print aesthetic.

The Code

<article class="magazine-content">
  <h2>The Art of Typography</h2>
  <p class="lead">Magazine layouts...</p>
  <p>Long form content...</p>
</article>
.magazine-content {
  column-count: 3;
  column-gap: 2.5rem;
  column-rule: 1px solid #e5e7eb;
  text-align: justify;
  hyphens: auto;
}

.magazine-content h2,
.magazine-content .lead {
  column-span: all;
}

@media (max-width: 900px) {
  .magazine-content {
    column-count: 2;
  }
}

@media (max-width: 600px) {
  .magazine-content {
    column-count: 1;
  }
}

Key Rules

  • column-count: 3 splits content into 3 equal columns automatically
  • column-gap controls space between columns — use rem for scale
  • column-rule adds a vertical divider without extra HTML
  • column-span: all forces headlines to break across all columns
  • hyphens: auto + text-align: justify = print-quality text flow
  • column-width alternative: columns: 18rem auto-calculates count
  • Use media queries to reduce column-count on smaller screens
  • Avoid height on container — let content determine column height
