Code 52: Live Demo with Zero HTML

Inject visual demos using SCSS only - no extra markup

What it does

This layout demonstrates how to inject a visual demo using only SCSS. No extra HTML markup needed for the demo box itself.

Uses ::before / ::after and SCSS variables/mixins to generate UI from empty hooks.

This layout demonstrates how to inject a visual demo using only SCSS. The colored box below is rendered entirely from .demo-hook using pseudo-elements.

↑ The demo box is generated from an empty div via SCSS

The Code

<div class="demo-hook"><span></span></div>
// code52.scss
$demo-color: #3b82f6;

@mixin demo-box {
  content: '';
  display: block;
  width: 100%;
  height: 200px;
  background: $demo-color;
  border-radius: 8px;
}

.demo-hook::before {
  @include demo-box;
}

Key Rules

  • ::before and ::after let you inject content without HTML
  • content: '' is required or pseudo-element won't render
  • SCSS @mixin reuses demo styles across multiple hooks
  • Use $variables for colors/sizes to keep demos consistent
  • display: block on pseudo-elements to set width/height
  • Keep hook divs empty — <span></span> only if you need multiple layers
  • Great for docs, styleguides, and component libraries
