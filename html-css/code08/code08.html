Code 8: Movie Poster Card

This project creates a movie poster card that reveals the synopsis, rating, and details when you hover over it.

What it does

It uses position: absolute for overlay, opacity + transition for smooth reveal, and transform for scale effects. No JavaScript needed.

Hover over any poster to see details

The Code

<article class="movie-card">
  <img src="poster.jpg" alt="Movie Poster">
  <div class="movie-overlay">
    <h3>Interstellar</h3>
    <p class="synopsis">A team of explorers...</p>
  </div>
</article>

Key Rules

  • Use position: relative on card + position: absolute on overlay
  • opacity: 0 to hide overlay, opacity: 1 on :hover to reveal
  • transition: opacity 0.3s ease for smooth fade effect
  • transform: scale(1.05) on image for zoom effect on hover
  • aspect-ratio: 2/3 keeps poster proportions consistent
