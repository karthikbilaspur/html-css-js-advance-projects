Code 25: Tabs Component

Radio button + :checked for tab switching - zero JavaScript

What it does

This tabs component uses hidden radio buttons and :checked to switch content panels. The label acts as the tab button.

Only one tab can be active at a time because all radios share the same name. Fully keyboard accessible — use Tab + Arrow keys or Space.

The Code

<input type="radio" id="tab1" name="tabs" checked>
<input type="radio" id="tab2" name="tabs">

<label for="tab1">Tab 1</label>
<label for="tab2">Tab 2</label>

<div class="tab-panel">Content 1</div>
<div class="tab-panel">Content 2</div>
/* Show panel when its radio is checked */
#tab1:checked ~ .tab-content #panel1,
#tab2:checked ~ .tab-content #panel2 {
  display: block;
}

Key Rules

  • All radios must have the same name attribute to form a group
  • Hide radios with display: none or position: absolute; opacity: 0
  • Use label for="id" to make labels clickable
  • Hide all .tab-panel by default, show only with #tab1:checked ~ .tab-content #panel1
  • Style the active label with #tab1:checked ~ .tab-labels label[for="tab1"]
  • Order matters: radios must come before content in HTML for ~ selector to work
