Code 41: Hero Section

Full-width banner with centered CTA - pure CSS

What it does

This is your hero section, the first thing visitors see. Full-viewport height with centered content, gradient overlay, and a clear call-to-action button.

The foundation of every landing page. Sets the tone and drives conversions.

↑ Full-width hero. Resize to test responsive typography

The Code

<section class="hero">
  <div class="hero-content">
    <h1>Build Amazing Experiences</h1>
    <p>Subheading text</p>
    <button class="cta-btn">Get Started</button>
  </div>
</section>
.hero {
  min-height: 100vh;
  display: flex;
  align-items: center;
  justify-content: center;
  text-align: center;
  background: linear-gradient(rgba(0,0,0,0.5), rgba(0,0,0,0.5)), 
              url('bg.jpg') center/cover;
}

.hero-content {
  max-width: 800px;
  padding: 2rem;
}

Key Rules

  • min-height: 100vh makes hero fill viewport height
  • display: flex + align-items: center + justify-content: center centers content
  • Use background: linear-gradient() overlay for text contrast on images
  • background-size: cover ensures image fills without distortion
  • Wrap text in .hero-content with max-width to prevent long lines
  • Use clamp(2rem, 5vw, 4rem) for responsive h1 sizing
  • Add text-shadow if background image is busy
