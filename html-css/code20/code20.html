Code 20: Toggle Switch

iOS-style toggles with checkbox hack and smooth CSS transitions

What it does

This project builds iOS-style toggle switches using only HTML + CSS with the checkbox hack.

It demonstrates input[type="checkbox"] hidden with appearance: none, :checked pseudo-class, and ::before/::after for the sliding effect.

The Code

<label class="toggle-switch">
  <input type="checkbox">
  <span class="toggle-slider"></span>
</label>

.toggle-switch input {
  display: none;
}

.toggle-switch input:checked + .toggle-slider {
  background-color: #34c759;
}

.toggle-slider::before {
  transform: translateX(26px);
}

Key Rules

  • Hide checkbox with display: none or opacity: 0
  • :checked + .toggle-slider - Styles track when checked
  • ::before pseudo-element - Creates the sliding circle
  • transform: translateX() - Moves circle from left to right
  • transition: all 0.3s ease - Smooth slide animation
  • Wrap in <label> - Makes entire switch clickable, improves accessibility
