Code 2: Profile Card

This project builds a centered profile card with avatar, name, bio, and social links using only HTML + CSS.

What it does

It demonstrates Flexbox centering, border-radius: 50% for circular images, and CSS transition for hover effects.

Live Demo

Jennie Smith

Frontend Developer

Building beautiful websites with HTML & CSS. Learning every day

GitHub LinkedIn Twitter

The Code

<article class="profile-card">
  <img src="avatar.jpg" alt="User Avatar" class="avatar">
  <h3>Jennie Smith</h3>
  <p class="role">Frontend Developer</p>
  <p class="bio">Building beautiful websites...</p>
  <div class="socials">
    <a href="#">GitHub</a>
    <a href="#">LinkedIn</a>
  </div>
</article>

Key Rules

  • Use display: flex + justify-content: center to center the card
  • border-radius: 50% makes any square image circular
  • box-shadow adds depth without images
  • transition on links/buttons gives smooth hover effects
  • Keep max-width on card so it looks good on mobile
