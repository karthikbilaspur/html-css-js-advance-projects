Code 11: Survey Form

Custom styled inputs, radio buttons, checkboxes, and range sliders with pure CSS

What it does

This project builds a styled survey form with multiple input types: text, email, radio, checkbox, select, textarea, and range.

It demonstrates custom styling for form controls using appearance: none, CSS pseudo-elements, and :checked state styling.

The Code

<form class="survey-form">
  <label for="name">Name</label>
  <input type="text" id="name" required>
  
  <label>
    <input type="radio" name="exp" value="beginner">
    Beginner
  </label>
  
  <input type="range" min="1" max="10" value="5">
  <button type="submit">Submit</button>
</form>

Key Rules

  • Use <label for="id"> to connect labels to inputs for accessibility
  • appearance: none removes default browser styling from inputs
  • input:checked + span styles custom radio/checkbox when selected
  • required attribute enables HTML5 validation
  • input[type="range"]::-webkit-slider-thumb styles the slider handle
  • Group related inputs with <fieldset> + <legend> for complex forms
