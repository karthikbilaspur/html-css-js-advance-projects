Code 24: <time>

The <time> element represents a specific period in time. Use datetime attribute for machine-readable format.

What it does

<time> marks up dates, times, and durations so browsers and search engines understand them.

Humans see your text. Machines read the datetime attribute.


The Code

HTML 
<p>Published on <time datetime="2026-06-10">June 10, 2026</time>.</p>
<p>The meeting is at <time datetime="15:30">3:30 PM</time>.</p>
<p>Reading time: <time datetime="PT5M">5 minutes</time>.</p>

Live Example

Published on June 10, 2026.

The meeting is at 3:30 PM.

Event runs Dec 24 to Dec 26, 2026.

Article reading time: 5 minutes.


datetime Format

time usage: The datetime attribute should be in ISO 8601 format:

Format Example Meaning
YYYY-MM-DD 2026-06-10 Date only
HH:MM 15:30 Time only
YYYY-MM-DDTHH:MM 2026-06-10T15:30 Date + time
PT5M PT5M Duration: 5 minutes

Helps browsers, calendars, and search engines understand dates.
