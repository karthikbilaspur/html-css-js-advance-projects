Code 73: <tr>

The <tr> element defines a row of cells in a table. It must be inside <table>, <thead>, <tbody>, or <tfoot>.

Examples

Basic tr with td

Row 1, Cell 1 Row 1, Cell 2
Row 2, Cell 1 Row 2, Cell 2

tr with th headers

Subject Marks Grade
Math 92 A
Science 88 B+

tr inside thead and tbody

Product Stock
Laptop 15
Mouse 42

tr with mixed th and td

Q1 ₹10,000 ₹12,000
Q2 ₹15,000 ₹18,000

Empty tr

Name Status
Server 1 Online
Server 2 Offline
Empty tr creates empty row

Note

tr rules: <tr> can only contain <th> and <td> elements. No text directly inside <tr>. Must be inside a table section.

Valid parents: <table>, <thead>, <tbody>, <tfoot>. Browsers auto-insert <tbody> if you skip it.

Row headers: Use <th scope="row"> inside <tr> to label the row. Helps screen readers announce row context for each cell.

No nesting: You cannot put <tr> inside another <tr>. For nested data, use a new table inside a <td>.
