Code 32: <ol> Ordered List

The <ol> element represents an ordered list of items. Sequence matters — usually rendered with numbers.

What it does

<ol> is for steps, rankings, instructions — anything where 1, 2, 3 matters.

Changing the order changes the meaning. That’s the semantic difference from <ul>.


The Code

HTML 
<ol>
  <li>Preheat oven to 180°C</li>
  <li>Mix dry ingredients</li>
  <li>Add wet ingredients</li>
  <li>Bake for 25 minutes</li>
</ol>

ol Attributes

ol attributes: start="5" sets starting number. reversed counts down. type="1|a|A|i|I" changes numbering style.

Attribute Example Result
start="5" <ol start="5"> 5, 6, 7...
reversed <ol reversed> 3, 2, 1
type="A" <ol type="A"> A, B, C...
type="i" <ol type="i"> i, ii, iii...

Use <ol> when order matters — instructions, rankings, processes. Use <ul> when it doesn't.
