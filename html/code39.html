Code 39: <figure> Figure

The <figure> element represents self-contained content, often with a caption using <figcaption>. Used for images, diagrams, code blocks, etc.

What it does

<figure> says “this content is a unit.” It can be moved away from the main flow and still make sense.

Think illustrations, code samples, charts, or quotes that you’d reference like “See Fig. 1”.


The Code

HTML 
<figure>
  <img src="chart.png" alt="Bar chart showing HTML tag usage">
  <figcaption>Fig. 1 — Most used HTML tags in 2026.</figcaption>
</figure>

figure vs img

figure vs img: Use <img> alone for inline images. Use <figure> when the image is referenced from the main content and has a caption.

<figcaption> must be first or last child of <figure>.

Not just images: <figure> can wrap code, video, audio, tables, or diagrams — anything that’s self-contained and captioned.
