Code 60: <textarea>

The <textarea> element represents a multi-line plain-text editing control. Use it for comments, messages, or long-form input.

Note

textarea rules: Unlike <input>, <textarea> has a closing tag. Default content goes between tags, not in value attribute.

Key attributes: rows sets visible lines, cols sets visible width. maxlength limits characters. placeholder shows hint text. required forces input.

Wrap behavior: wrap="hard" inserts line breaks on submit when text wraps. Requires cols. wrap="soft" is default — visual wrap only. wrap="off" no wrapping.

Form submit: Unlike <input>, newlines in <textarea> are preserved as \n in submitted data. Use readonly to display but prevent editing.
