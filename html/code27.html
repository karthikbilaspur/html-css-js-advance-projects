Code 27: <sub> Subscript

The <sub> element represents subscript text — text that should be displayed lower and smaller than the surrounding text.

What it does

<sub> is for chemical formulas, mathematical notation, and footnotes.

It’s semantic — it means “this character goes below the baseline” for a reason, not just for looks.


The Code

HTML 
<p>The chemical formula for water is H<sub>2</sub>O.</p>
<p>Carbon dioxide is CO<sub>2</sub>.</p>
<p>In math: log<sub>2</sub> 8 = 3</p>

Live Example

The chemical formula for water is H2O.

Carbon dioxide is CO2.

Glucose: C6H12O6

In math: log2 8 = 3

Footnotes are sometimes marked with numbers like this1.


sub vs CSS

sub usage: Use <sub> for typographical conventions like chemical formulas, mathematical variables, and footnotes.

Don't use it just to lower text visually — use CSS vertical-align: sub for that. The semantic meaning matters for screen readers and search engines.
