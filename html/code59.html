Code 59: <input type="date">

The <input type="date"> element creates a date picker for selecting year, month, and day. Value format is always YYYY-MM-DD.

Note

date type rules: Browser shows calendar picker. Value always submits as YYYY-MM-DD regardless of display locale.

Key attributes: min and max limit selectable dates. step sets day intervals — step="7" for weekly. value sets default.

Validation: Invalid dates blocked. Empty is valid unless required. min/max violations prevent submit.

Fallback: If unsupported, degrades to type="text". Use pattern="\d{4}-\d{2}" as backup. For date + time together, use type="datetime-local".
