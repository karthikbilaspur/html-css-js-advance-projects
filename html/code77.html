Code 77: <tbody>

The <tbody> element groups the body content in a table. It contains the main data rows after <thead> and before <tfoot>.

Note

tbody rules: <tbody> contains <tr> elements with main table data. Must be inside <table>, after <thead> and <caption>, before <tfoot>.

Multiple tbody: A table can have multiple <tbody> elements to group rows. Useful for separating data sets like years or categories.

Optional but recommended: If you skip <tbody>, browsers insert one automatically. Explicit <tbody> makes structure clear and helps CSS/JS targeting.

Order matters: DOM order is <thead>, <tbody>, <tfoot> regardless of source order. Browser renders <tfoot> at bottom.
