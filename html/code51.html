Code 51: <form>

The <form> element represents a document section containing interactive controls for submitting information.

Note

form rules: <form> groups controls like <input>, <button>, <select>. Forms cannot be nested inside other forms.

Key attributes: action sets where data is sent. method is get for search/bookmarkable data, post for sensitive data. name on inputs becomes the key in submitted data.

Validation: required, min, max, pattern on inputs enable browser validation. Add novalidate to disable it.

UX: autocomplete="on" helps users fill forms faster. Use <label for="id"> to link labels to inputs for accessibility.
