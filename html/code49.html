Code 49: <canvas>

The <canvas> element provides a resolution-dependent bitmap canvas for rendering graphs, game graphics, art, or other visual images on the fly using JavaScript.

Note

canvas rules: <canvas> is just a container. It has no content or border by default. You must use JavaScript to draw on it via the Canvas 2D API or WebGL.

Key attributes: width and height set the coordinate space in pixels. Defaults to 300x150 if omitted. This is different from CSS width/height which scales the bitmap.

Fallback content: Content inside <canvas> shows if JS is disabled or canvas isn't supported. Use it for accessibility or static image alternatives.

Not accessible by default: Canvas drawings are bitmaps. Screen readers can't parse them. Provide text alternatives or ARIA descriptions for complex visuals.
