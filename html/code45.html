Code 45: <iframe>

The <iframe> element embeds another HTML page within the current page. Creates a nested browsing context.

Examples

Note

Key attributes: src sets the URL to embed. width and height control size. title is required for accessibility — describes the frame content.

Security: sandbox restricts what the iframe can do. Values: allow-scripts, allow-same-origin, allow-forms. Empty sandbox = maximum restrictions.

Performance: loading="lazy" defers offscreen iframes. allow controls features like camera, microphone, autoplay. allowfullscreen enables fullscreen.

Fallback: Content between <iframe> tags shows if iframes aren't supported.
