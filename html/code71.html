Code 71: <meter>

The <meter> element represents a scalar measurement within a known range, or a fractional value. Use for gauges, not progress.

Examples

Basic meter


6 out of 10
6 of 10

Meter with low, high, optimum


80%
80% used — above high range

Meter with optimum high


85
85 — in high range, near optimum

Meter with optimum low


20%
20% — low is good here

Meter without optimum


40%
40% — in medium range

Meter with decimal


4.2 out of 5

Meter updated with input


22°C

Note

meter vs progress: Use <meter> for static measurements like battery, grades, disk space. Use <progress> for task completion like file upload.

Key attributes: value is current. min and max set range. low and high define ranges. optimum is the preferred value.

Color regions: Browsers color the bar based on where value falls relative to low, high, and optimum. No CSS needed for green/yellow/red.

Fallback content: Text inside <meter>6 out of 10</meter> shows in old browsers and screen readers. Always include it.

Don't use for: Don't use <meter> for percentages that change over time. That's <progress>. Meter is for known, static values.
