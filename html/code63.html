Code 63: <optgroup>

The <optgroup> element groups related <option> elements inside a <select>. Creates non-selectable headings for better organization.

Note

optgroup rules: Must be inside <select>. Cannot be nested. The label attribute is required — it displays as the group heading.

Key behavior: <optgroup> itself is not selectable. Only the <option> elements inside it can be chosen. Groups options visually in the dropdown.

Disabled groups: <optgroup disabled> greys out the whole group. All child options become unselectable.

Accessibility: Screen readers announce the group label when navigating options. Use optgroup to organize long lists like countries by continent, products by brand.
