Code 69: <output>

The <output> element represents the result of a calculation or user action. It links to inputs via the for attribute.

Note

output purpose: Use <output> specifically for calculated results from user input. For static text use <span> or <p>.

Key attributes: for takes space-separated input ids it depends on. name includes it in form submission. form links it to a form by id.

Live updates: Use oninput on the form or inputs to update output.value instantly as user types or changes values.

Accessibility: Screen readers announce <output> changes when related inputs update. The for attribute provides context about which inputs affect it.
