Code 33: <li> List Item

The <li> element represents an item in a list. It must be a direct child of <ul>, <ol>, or <menu>.

What it does

<li> is the building block of every list. No <ul> or <ol> is complete without it.

It’s a container — it can hold text, images, headings, paragraphs, links, or even nested lists.


The Code

HTML 
<ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>

<ol>
  <li value="10">This starts at 10</li>
  <li>Next is 11</li>
</ol>

li Rules

li rules: <li> only works inside <ul>, <ol>, or <menu>.

The value attribute only works in <ol> to set a specific number. An <li> can contain any flow content — paragraphs, headings, images, even other lists.
