Code 43: <source>

The <source> element specifies multiple media resources for <video>, <audio>, or <picture>. Browser picks the first supported format.

Examples

Video with multiple formats

Audio with multiple formats

Responsive images with picture

Video with media queries


Note

source rules: <source> is a void element — no closing tag. Must be child of <video>, <audio>, or <picture>. Browser uses the first <source> it can play.

Key attributes: src sets the file. type helps browser skip unsupported formats faster: video/mp4, audio/mpeg. media adds media query conditions. srcset for responsive images in <picture>.
