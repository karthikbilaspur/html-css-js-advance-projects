Code 79: <caption>

The <caption> element represents the title of a table. It must be the first child of <table> and provides an accessible name.

Examples

Basic caption

Employee Salary Data
Name Role Salary
Vikram Developer ₹80,000
Shreya Designer ₹70,000

Caption with description

Q1 2026 Sales Report
All figures in INR
Region Revenue
South ₹12,50,000
North ₹9,80,000

Caption with link

Temperature Data - Source: IMD
City Max Min
Bangalore 32°C 21°C

Caption placement

Correct: caption is first child of table
Item Stock
Laptop 15

Note

caption rules: Must be first child of <table>, before <thead> or <tbody>. Only one <caption> per table.

Purpose: Gives table an accessible name. Screen readers announce caption first, so users know what the table is about before hearing cells.

Content: Can contain text, <small>, <a>, <strong>. Keep it concise but descriptive. Not for styling — that’s CSS.

caption vs summary: <caption> is visible title. The old summary attribute is obsolete. Use caption + surrounding text for description.

Placement matters: If you put caption after <thead>, it’s invalid HTML. Always first inside <table>.
