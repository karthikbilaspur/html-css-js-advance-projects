Code 44: <track>

The <track> element specifies timed text tracks for <video> and <audio> elements. Used for subtitles, captions, descriptions, chapters.

Examples

Video with subtitles

Captions vs subtitles

Chapters and descriptions


Note

track rules: <track> is a void element — no closing tag. Must be child of <video> or <audio>. Uses WebVTT .vtt files.

kind values: subtitles = dialogue translation. captions = dialogue + sound effects for deaf users. descriptions = audio descriptions for blind users. chapters = navigation. metadata = data for JS.

Key attributes: src points to .vtt file. srclang sets language code like en. label shows in UI. default enables it by default.
