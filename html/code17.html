Code 17: <blockquote> Block Quotation

The <blockquote> element represents content quoted from another source. Use cite attribute for the source URL.

What it does

<blockquote> is for long quotations that need their own block. Browsers usually indent it.

It tells search engines and screen readers "this text is from someone else."


The Code

HTML 
<blockquote cite="https://www.w3.org/TR/html/">
  <p>The World Wide Web's markup language has always been HTML.</p>
</blockquote>

<blockquote>
  <p>Talk is cheap. Show me the code.</p>
  <footer>— <cite>Linus Torvalds</cite></footer>
</blockquote>

Live Example

The World Wide Web's markup language has always been HTML.

Talk is cheap. Show me the code.

— Linus Torvalds

blockquote vs q

blockquote vs q: Use <blockquote> for long quotations that need their own block. Use <q> for short inline quotes.

<blockquote> = block-level. <q> = inline.
