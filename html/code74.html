Code 74: <td>

The <td> element defines a standard data cell in a table. It must be inside a <tr> element.

Note

td rules: <td> is for data cells. Use <th> for headers. Must be direct child of <tr>.

Key attributes: colspan merges cells horizontally. rowspan merges vertically. headers links to <th> ids for accessibility.

Content: Can contain text, images, links, lists, even nested tables. Avoid block elements like <div> unless needed.

td vs th: Use <td> for data, <th> for labels. Browsers bold <th> and center it by default. Screen readers treat them differently.
