Code 21: <pre> Preformatted Text

The <pre> element represents preformatted text. Whitespace, line breaks, and spacing are preserved exactly as written.

What it does

<pre> tells the browser "display this exactly as typed." Spaces, tabs, and line breaks are kept.

It defaults to a monospace font, making it perfect for code, ASCII art, and aligned text.


The Code

pre vs code

pre vs code: <pre> preserves whitespace and uses a monospace font. <code> is for marking code but doesn't preserve formatting alone.

Combine them: <pre><code>...</code></pre> for proper code blocks with semantic meaning.
