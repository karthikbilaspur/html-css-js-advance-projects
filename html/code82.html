Code 82: <details>

The <details> element creates a disclosure widget. Content is hidden by default and revealed when toggled. Use with <summary>.

Note

details rules: First child should be <summary>. Everything after summary is hidden until toggled. No JS needed — it's native.

summary element: Acts as the clickable label. If missing, browser inserts "Details" text. Always write your own summary for clarity.

open attribute: Add open to <details open> to expand by default. Users can still collapse it.

Accessibility: Screen readers announce it as disclosure widget. Keyboard users can toggle with Space or Enter when summary is focused.

Content: Can contain any flow content — paragraphs, lists, tables, images. Perfect for FAQs, spoilers, or progressive disclosure.
