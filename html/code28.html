Code 28: <sup> Superscript

The <sup> element represents superscript text — text that appears higher and usually smaller than the main text line.

What it does

<sup> is for exponents, ordinal numbers, footnotes, and trademark symbols.

Like <sub>, it’s semantic — use it for typographical meaning, not visual tricks.


The Code

HTML 
<p>Einstein's famous equation: E = mc<sup>2</sup></p>
<p>Today is June 10<sup>th</sup>, 2026.</p>
<p>Footnotes use superscript: This fact<sup>1</sup></p>

sup vs sub

sup vs sub: Use <sup> for exponents, ordinal indicators, footnotes, and trademarks.

Use <sub> for subscript like in H2O. Only use these for typographical conventions, not just to raise text — that's CSS vertical-align.

Tag Position Common Uses
<sup> Above baseline E=mc2, 1st, ™, footnotes1
<sub> Below baseline H2O, log2, C6H12O6
