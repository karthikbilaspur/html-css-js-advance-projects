Code 38: <img> Image

The <img> element embeds an image into the document. It's a void element — no closing tag.

What it does

<img> is how you put pictures on the web. Period.

Void element = self-closing. src and alt are the only two attributes you truly need. Everything else is optimization.


The Code

HTML 
<img src="logo.png" alt="HTML 100 Codes Vault logo">

<img src="banner.jpg" alt="Welcome banner" width="600" height="200">

<img src="divider.png" alt="">

img Rules

Required attributes: src specifies the image URL. alt provides text description for screen readers and when the image fails to load.

If the image is purely decorative, use alt="". Never omit alt — that fails accessibility.

Best practices: Always include alt. Use width and height to prevent layout shift. For responsive images, look at srcset and <picture>.

Void element: <img> has no closing tag. Never write </img>.
