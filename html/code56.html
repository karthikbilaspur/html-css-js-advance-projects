Code 56: <input type="checkbox">

The <input type="checkbox"> element creates a checkbox control that can be toggled on or off. Used for single or multiple selections.

Note

checkbox rules: Only sends data if checked. If unchecked, the name/value pair is omitted from submission entirely.

Key attributes: checked makes it on by default. required forces user to check it. value is what gets submitted — defaults to "on" if omitted.

Grouping: Multiple checkboxes with same name submit as array: skills=html&skills=css. Use <fieldset> + <legend> to group related checkboxes.

States: Has three states: checked, unchecked, and indeterminate. Indeterminate only set via JS — used for "select all" when some children are checked.
