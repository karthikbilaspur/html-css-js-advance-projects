Code 84: <dialog>

The <dialog> element represents a modal or non-modal dialog box. Use JavaScript showModal() or show() to display it.

Note

dialog methods: Use showModal() for modal that blocks page. Use show() for non-modal. Use close() to hide it.

open attribute: Makes dialog visible but not modal. Rarely used — control with JS instead for better UX.

form method="dialog": Buttons inside close the dialog automatically. Button value becomes dialog.returnValue.

Accessibility: showModal() traps focus, handles ESC to close, and sets proper ARIA roles. Don't build modals with divs when dialog exists.

Backdrop: Modal dialogs have a ::backdrop pseudo-element you can style with CSS. Not shown here since this is HTML-only.
