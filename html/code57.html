Code 57: <input type="radio">

The <input type="radio"> element creates a radio button for selecting one option from a group. Only one radio with the same name can be selected.

Note

radio rules: Radio buttons with the same name form a group. Only one can be checked. Selecting a new one unchecks the previous.

Key attributes: name groups radios together. value is what submits. checked sets default. required on any radio in group forces user to pick one.

Unlike checkbox: Radios = choose one only. Checkboxes = choose zero to many. Once a radio is selected in a group, it can't be unselected except by choosing another.

Accessibility: Always use <fieldset> + <legend> to group. Screen readers announce the group name when focusing any radio. Each needs its own <label for="id">.
