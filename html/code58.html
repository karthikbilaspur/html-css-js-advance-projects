Code 58: <input type="file">

The <input type="file"> element lets users select one or more files from their device to upload to a server.

Note

file input rules: Must use method="post" and enctype="multipart/form-data" on the form. GET cannot upload files.

Key attributes: accept hints file types like image/*, .pdf, .doc,.docx. multiple allows several files. capture opens camera: user for selfie, environment for rear.

Security: value cannot be set via HTML/JS for privacy. Browser shows filename only. Path is hidden — you get C:\fakepath\file.jpg.

Validation: accept is only a hint. Users can bypass it. Always validate file type and size on server. Use required to force selection.
