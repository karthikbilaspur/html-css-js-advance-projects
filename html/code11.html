Code 11: <h1> to <h6> Headings

What it does

Heading tags define titles and subtitles. <h1> is most important, <h6> is least important.

Search engines and screen readers use headings to understand page structure.


The Code

HTML 
<h1>Main Page Title</h1>
<h2>Major Section</h2>
<h3>Subsection</h3>
<h4>Smaller heading</h4>
<h5>Even smaller</h5>
<h6>Smallest heading</h6>

Key Rules

  • Use only one <h1> per page
  • Don’t skip levels: don’t jump from <h1> to <h3>
  • Don’t use headings just to make text big — that’s what CSS is for
