Code 42: <video>

The <video> element embeds video content in documents. Use <source> for multiple formats and <track> for captions.

Examples

Basic video with controls

Multiple formats + poster

Autoplay muted + loop

With subtitles


Note

Key attributes: controls shows UI. poster displays image before play. autoplay usually needs muted to work. playsinline prevents iOS fullscreen. preload="none|metadata|auto" controls loading.

Formats: Use <source> with MP4 + WebM for cross-browser support. Fallback text inside <video> shows if unsupported.

Accessibility: Add <track kind="captions"> or subtitles. Set width and height to avoid layout shift.
