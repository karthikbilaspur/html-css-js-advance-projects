Code 37: <a> Anchor

The <a> element creates a hyperlink to other web pages, files, locations within the same page, email addresses, or any other URL.

What it does

<a> is the most important tag on the web. Without it, there’s no web — just isolated pages.

The href attribute defines the destination. No href = not a link, just an anchor placeholder.


The Code

HTML 
<a href="https://www.mozilla.org">Visit Mozilla</a>
<a href="index.html">Back to Home</a>
<a href="#note1">Jump to footnote</a>
<a href="mailto:hello@example.com">Email us</a>

Key Attributes

Key attributes: href sets the destination URL. target="_blank" opens in new tab — always pair with rel="noopener" for security.

download prompts file save instead of navigate. rel describes relationship like nofollow or author.

href Value Purpose
https://site.com External URL
/about.html Internal page
#section Jump to ID on page
mailto:email Open email client
tel:+123 Start phone call

Accessible links: Link text should make sense out of context. Avoid "click here" — use "Download the HTML cheatsheet" instead.
