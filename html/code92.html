Code 92: <meta name="description">

The description meta tag summarizes page content. Search engines often display it in results under your title. It doesn't affect rankings directly, but impacts click-through rate.

Examples

1. Basic product page

<meta name="description" content="Buy Wireless Ergonomic Mouse with 3-year warranty. Free shipping on orders over $50. Shop now at TechGear.">
Clear, action-oriented, includes key selling points

2. Blog article

<meta name="description" content="Master CSS Flexbox in 10 minutes. Learn justify-content, align-items, and flex-wrap with interactive examples and cheat sheet.">
Tells user what they'll learn and how long it takes

3. Homepage

<meta name="description" content="HTML 100 Codes Vault teaches HTML with 99 practical examples. From <div> to <canvas>, learn semantic tags, forms, and APIs. Free tutorials.">
Summarizes entire site value proposition

4. Bad vs Good

Bad - keyword stuffing:

<meta name="description" content="HTML HTML tutorial HTML learn HTML code HTML examples HTML CSS JavaScript">

Bad - too vague:

<meta name="description" content="Welcome to our website">

Good:

<meta name="description" content="Learn HTML with 99 hands-on code examples covering forms, semantic tags, and modern APIs. Start coding today.">

5. Local business

<meta name="description" content="Joe's Pizza in Bangalore - Wood-fired pizzas, pasta, and desserts. Open 11am-11pm. Dine-in, takeaway & Swiggy delivery. Call 080-12345678.">
Include location, hours, and contact for local SEO

6. Article with date

<meta name="description" content="Updated June 2026: Complete guide to HTML5 semantic elements. Learn when to use article vs section vs div with real examples.">
Date shows freshness. Google may display it

Note

Length: Aim for 150-160 characters. Google truncates around 155-160 on desktop, ~120 on mobile. Put the important stuff first.

Unique per page: Every page needs its own description. Duplicate metas hurt SEO. If you can't write unique ones, better to omit it than duplicate.

No keyword stuffing: Write for humans. Google ignores metas stuffed with keywords and may pick its own snippet from your page.

Call to action: Use active voice: "Learn", "Shop", "Download", "Discover". It improves click-through rate.

Not a ranking factor: Google said in 2009 that meta description doesn't affect rankings. But a good one gets more clicks, which indirectly helps.

Special characters: Quotes inside need escaping: content="Learn "HTML" fast". Avoid em dashes and weird symbols — they can break.

Google may ignore it: If Google thinks your page content has a better snippet for the search query, it will override your meta description. That's normal.

Test it: Search site:yourdomain.com on Google to see how your descriptions actually appear.
