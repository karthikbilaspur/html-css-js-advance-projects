Code 16: <em> Emphasized Text

The <em> tag represents stress emphasis. Screen readers will change inflection.

What it does

<em> marks text that should be spoken with emphasis. Browsers usually render it in italics.

Like <strong>, it’s semantic — it adds meaning, not just styling.


The Code

HTML 
<p>You <em>must</em> save your work.</p>
<p>I said I <em>like</em> HTML, not love it.</p>

Live Example

You must save your work.

I said I like HTML, not love it.

Cats are cute animals, not scary ones.


<em> vs <i>

em vs i: Use <em> for semantic emphasis. Use <i> only for things like foreign words, technical terms, or thoughts — not emphasis.

<em> = stress. <i> = alternate voice.
