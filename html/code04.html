Code 04: <meta charset="UTF-8">

What it does

The meta charset tag tells the browser which character encoding to use for the page.

UTF-8 handles pretty much every character: English, हिन्दी, 中文, emoji 😀, math symbols ∑.

Without it, you get weird symbols like � instead of real characters.


The Code

HTML 
<meta charset="UTF-8">

This tag always goes inside <head> and should be one of the first things there.


Live Example

With UTF-8 you can write: Hello, नमस्ते, 你好, مرحبا, ©, ™, €

If the charset was wrong, those would look broken.
