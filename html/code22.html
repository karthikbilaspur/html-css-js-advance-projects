Code 22: <abbr> Abbreviation

The <abbr> element represents an abbreviation or acronym. Use the title attribute to provide the full expansion.

What it does

<abbr> marks up shortened terms. The title attribute gives the full form.

Most browsers show a dotted underline. Hovering shows the tooltip. Screen readers announce the full term.


The Code

HTML 
<p><abbr title="HyperText Markup Language">HTML</abbr> is the standard markup language.</p>
<p>We use <abbr title="Cascading Style Sheets">CSS</abbr> to style documents.</p>

Live Example

HTML is the standard markup language for web pages.

We use CSS to style our documents.

Pro Tip

abbr usage: Always include title with the full term on first use. Screen readers and browsers use it to help users.

Don't use it for random shortenings — only for actual abbreviations/acronyms people might not know.
